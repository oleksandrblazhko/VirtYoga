# Сценарії використання математичного опису для IT-проєкту "WebcamPhysExercise"

## Лекція 1. «Матриці. Означення, операції над матрицями. Обернена матриця. Теорема про існування оберненої матриці. Метод елементарних перетворень обчислення оберненої матриці. Розв’язання систем лінійних неоднорідних алгебраїчних рівнянь матричним способом»

### Обгрунтування користі:
Використання матричного числення дозволяє ефективно обробляти та аналізувати координати суглобів людини, що виконує йога-вправи. Матриці забезпечують зручний спосіб представлення тривимірних даних (x, y, z) для кожного суглоба, що дозволяє легко виконувати перетворення, порівняння та аналіз пози тіла. Матричні операції є основою для багатьох алгоритмів комп'ютерного зору та аналізу руху.

### Зв'язок математичної формалізації із описом IT-проєкту:
- Матриці використовуються для представлення координат суглобів скелету людини, отриманих від бібліотеки Mediapipe
- Операції над матрицями дозволяють виконувати порівняння поточної пози з еталонною
- Обернені матриці можуть бути використані для обчислення необхідних трансформацій для досягнення правильної пози
- Системи лінійних рівнянь, що розв'язуються матричним способом, можуть моделювати зв'язки між суглобами

### Приклади математичного опису IT-проєкту:

**1. Представлення даних про суглоби:**
```
Нехай A - матриця розміру n×3, де n - кількість суглобів, а 3 - координати (x, y, z):

A = [x₁  y₁  z₁]
    [x₂  y₂  z₂]
    [⋮   ⋮   ⋮ ]
    [xn  yn  zn]

Ця матриця представляє поточну позу тіла з n суглобами.
```

**2. Представлення еталонної пози:**
```
Нехай A₀ - матриця еталонної пози для певної йога-вправи:

A₀ = [x₁₀  y₁₀  z₁₀]
     [x₂₀  y₂₀  z₂₀]
     [ ⋮    ⋮    ⋮ ]
     [xn₀  yn₀  zn₀]

Ця матриця містить ідеальні координати для виконання певної йога-пози.
```

**3. Обчислення відхилення:**
```
Матриця відхилень Δ = A - A₀, де кожен елемент Δᵢⱼ = aᵢⱼ - a₀ᵢⱼ
Це дозволяє визначити, наскільки поточна поза відрізняється від еталонної.
Норма матриці ||Δ|| може використовуватися як загальна міра відхилення.
```

**4. Визначення рекомендацій для корекції пози:**
```
Якщо ||Δ|| > ε (де ε - припустима похибка), система визначає напрямок
корекції шляхом аналізу окремих елементів матриці Δ.
Матриця корекції C може бути обчислена як C = A₀ - A
```

**5. Матричні перетворення для аналізу кутів між суглобами:**
```
Для аналізу кутів між суглобами можна використовувати вектори,
представлені як стовпці матриці, і застосовувати скалярний добуток:
cos(θ) = (u·v)/(|u||v|), де u, v - вектори між суглобами
```

**6. Обертання тіла для нормалізації пози:**
```
Матриця обертання R дозволяє обертати позу для порівняння:
A' = A × R, де R - ортогональна матриця обертання.
Це дозволяє нормалізувати позу незалежно від орієнтації тіла відносно камери.
```

**7. Масштабування для нормалізації розміру тіла:**
```
Матриця масштабування S дозволяє нормалізувати розмір тіла:
A'' = A' × S, де S - діагональна матриця масштабування.
Це дозволяє порівнювати пози незалежно від розміру тіла користувача.
```

**8. Розв'язання систем лінійних рівнянь для визначення суглобів:**
```
Система рівнянь Ax = b може моделювати зв'язки між суглобами,
де A - матриця коефіцієнтів, що визначає анатомічні обмеження,
x - вектор невідомих параметрів пози, b - вектор обмежень.
Розв'язок x = A⁻¹b дає оптимальні параметри пози.
```

**9. Використання визначника матриці для аналізу стабільності:**
```
Визначник матриці, складеної з координат ключових суглобів,
може використовуватися для аналізу стабільності пози тіла.
Якщо det(A) близький до 0, то суглоби майже лежать в одній площині.
```

**10. Елементарні перетворення для очищення даних:**
```
Елементарні перетворення матриці можуть використовуватися
для виявлення та усунення помилкових вимірювань суглобів,
що допомагає покращити точність аналізу пози.
```

Ці математичні моделі дозволяють системі автоматично аналізувати позу людини, порівнювати її з еталонною, визначати відхилення та надавати рекомендації щодо корекції рухів для досягнення правильної йога-пози.
