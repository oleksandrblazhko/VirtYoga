# Сценарії використання математичного опису для IT-проєкту "WebcamPhysExercise"

## Лекція 5. «Векторний та мішаний добуток. Означення. Властивості. Застосування векторного і мішаного добутків»

### Обгрунтування користі:
Векторний та мішаний добутки є потужними інструментами для аналізу просторових відносин між суглобами тіла. Вони дозволяють обчислювати площі трикутників, утворених суглобами, визначати орієнтацію площин, на яких лежать частини тіла, та аналізувати просторову компланарність суглобів, що є важливим для аналізу складних йога-поз. Ці операції є частиною векторної алгебри, яка є теоретичною основою аналітичної геометрії, що дозволяє описати будь-яку геометричну фігуру алгебраїчними засобами, що критично важливо для автоматичного аналізу пози тіла.

### Зв'язок математичної формалізації із описом IT-проєкту:
- Векторний добуток використовується для обчислення площі трикутника, утвореного трьома суглобами
- Мішаний добуток дозволяє визначати, чи лежать чотири суглоби в одній площині
- Векторний добуток використовується для знаходження нормалей до площин, утворених частинами тіла
- Векторний добуток дозволяє визначати напрямок обертання між векторами суглобів
- Мішаний добуток використовується для обчислення об'єму тіла, утвореного чотирма суглобами

### Приклади математичного опису IT-проєкту:

**1. Обчислення площі трикутника, утвореного трьома суглобами:**
```
Нехай A(x₁, y₁, z₁), B(x₂, y₂, z₂), C(x₃, y₃, z₃) - координати трьох суглобів.
Тоді вектори AB = (x₂-x₁, y₂-y₁, z₂-z₁) і AC = (x₃-x₁, y₃-y₁, z₃-z₁).
Векторний добуток AB × AC = (ay×cz - az×cy, az×cx - ax×cz, ax×cy - ay×cx)
де ax = x₂-x₁, ay = y₂-y₁, az = z₂-z₁, cx = x₃-x₁, cy = y₃-y₁, cz = z₃-z₁

Площа трикутника ABC = (1/2) × |AB × AC|
Це дозволяє аналізувати площу, яку займає певна частина тіла в просторі.
```

**2. Визначення нормалі до площини, утвореної трьома суглобами:**
```
Векторний добуток AB × AC дає вектор, перпендикулярний до площини, що містить точки A, B, C.
Це дозволяє визначити орієнтацію площини, на якій лежать певні частини тіла (наприклад, площина тулуба або площа, утворена розведеними руками).
```

**3. Визначення компланарності чотирьох суглобів за допомогою мішаного добутку:**
```
Нехай A, B, C, D - чотири суглоби. Вектори AB, AC, AD компланарні (лежать в одній площині),
якщо мішаний добуток (AB × AC) · AD = 0.

Мішаний добуток обчислюється як визначник:
| (x₂-x₁) (y₂-y₁) (z₂-z₁) |
| (x₃-x₁) (y₃-y₁) (z₃-z₁) | = 0
| (x₄-x₁) (y₄-y₁) (z₄-z₁) |

Якщо визначник дорівнює 0, то суглоби лежать в одній площині.
```

**4. Визначення напрямку обертання між векторами:**
```
Знак векторного добутку AB × AC вказує на напрямок обертання від вектора AB до вектора AC.
Це може бути корисним для аналізу напрямку руху під час переходу між позами.
```

**5. Обчислення об'єму тетраедра, утвореного чотирма суглобами:**
```
Якщо чотири суглоби не лежать в одній площині, вони утворюють тетраедр.
Об'єм тетраедра ABCD = (1/6) × |(AB × AC) · AD|

Це дозволяє аналізувати просторову конфігурацію тіла в тривимірному просторі.
```

**6. Визначення кута між векторами за допомогою векторного добутку:**
```
Кут φ між векторами AB і AC можна знайти за формулою:
sin(φ) = |AB × AC| / (|AB| × |AC|)

Це дозволяє обчислювати кути між частинами тіла, що важливо для аналізу йога-поз.
```

**7. Перевірка колінеарності векторів:**
```
Два вектори AB і CD колінеарні (паралельні), якщо їх векторний добуток дорівнює нулю:
AB × CD = 0

Це може бути корисним для аналізу, чи певні частини тіла (наприклад, дві руки) 
розташовані паралельно, як передбачено в певній йога-позі.
```

**8. Визначення площі паралелограма, утвореного двома векторами суглобів:**
```
Площа паралелограма, утвореного векторами AB і AC, дорівнює |AB × AC|.
Це може бути корисним для аналізу площі, яку займає певна частина тіла,
наприклад, площі, утвореної розведеними руками.
```

**9. Використання мішаного добутку для аналізу об'єму тіла:**
```
Мішаний добуток (AB × AC) · AD може використовуватися для визначення об'єму,
який займає певна частина тіла, що важливо для аналізу просторової конфігурації
складних йога-поз.
```

**10. Визначення орієнтації трикутника суглобів:**
```
Знак мішаного добутку (AB × AC) · AD відносно зовнішньої точки D
може вказувати на орієнтацію трикутника ABC (за або проти годинникової стрілки),
що важливо для аналізу симетрії та правильного виконання певних поз.
```

**11. Використання векторного добутку для визначення моменту сили:**
```
У контексті аналізу балансу тіла, векторний добуток може використовуватися
для обчислення моменту сили, що діє на певну частину тіла відносно суглоба.
Це важливо для аналізу стабільності йога-поз.
```

**12. Визначення площі проекції трикутника на площину:**
```
Площа проекції трикутника ABC на площину з нормальним вектором n
дорівнює |(AB × AC) · n| / |n|.
Це може бути корисним для аналізу, як частина тіла виглядає з певного ракурсу.
```

Ці математичні моделі дозволяють системі аналізувати просторові відносини між суглобами, визначати площі, об'єми, орієнтацію частин тіла та перевіряти, чи відповідає поза людини еталонній моделі йога-пози.
