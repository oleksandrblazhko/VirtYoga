# Сценарії використання математичного опису для IT-проєкту "WebcamPhysExercise"

## Лекція 6.  «Пряма на площині: загальне рівняння, нормальне, з кутовим коефіцієнтом, через дві точки, у відрізках, канонічне рівняння, через точку перпендикулярно вектору. Кут між прямими. Відстань від точки до прямої. Площина: загальне рівняння, нормальне, рівняння у відрізках, рівняння через точку перпендикулярно вектору, через три точки. Відстань від точки до площини. Умови паралельності і перпендикулярності площин»

### Обгрунтування користі:
Аналітична геометрія дозволяє точно моделювати лінійні та площинні елементи пози тіла людини. Це особливо важливо для аналізу правильності розташування частин тіла, таких як руки, ноги, тулуб, а також для визначення відстаней між елементами пози та еталонними лініями або площинами. Використання координатного методу дозволяє описати будь-яку геометричну фігуру алгебраїчними засобами, що критично важливо для автоматичного аналізу пози тіла. Метод координат, запропонований Рене Декартом, дозволяє визначити положення фігури в R1, R2 чи R3, що істотно для аналізу 3D-пози тіла людини.

### Зв'язок математичної формалізації із описом IT-проєкту:
- Рівняння прямої використовується для моделювання ліній, утворених послідовними суглобами (наприклад, лінія руки або ноги)
- Рівняння площини застосовується для аналізу площин, на яких мають лежати певні частини тіла
- Відстань від точки до прямої/площини дозволяє визначати відхилення суглобів від еталонної пози
- Кут між прямими використовується для аналізу кутів між частинами тіла
- Нормальні вектори використовуються для визначення орієнтації частин тіла
- Канонічні рівняння прямої застосовуються для точного визначення напрямку ліній суглобів
- Рівняння прямої у відрізках використовується для аналізу обмежень руху по осях

### Приклади математичного опису IT-проєкту:

**1. Моделювання лінії між суглобами за допомогою канонічного рівняння прямої:**
```
Нехай A(x₁, y₁, z₁) і B(x₂, y₂, z₂) - координати двох послідовних суглобів (наприклад, плече і лікоть).
Канонічне рівняння прямої, що проходить через ці точки в 3D просторі:
(x-x₁)/(x₂-x₁) = (y-y₁)/(y₂-y₁) = (z-z₁)/(z₂-z₁)

де (x₂-x₁, y₂-y₁, z₂-z₁) - координати напрямного вектора s = (m, n, p).
Це дозволяє моделювати положення кінцівок та перевіряти, чи вони відповідають еталонній лінії пози.
```

**2. Визначення відстані від суглоба до прямої лінії:**
```
Якщо в еталонній позі передбачено, що певний суглоб C(x₀, y₀, z₀) має лежати на певній лінії,
заданій точками A(x₁, y₁, z₁) і B(x₂, y₂, z₂), то відстань від точки C до цієї лінії:
d = |AC × AB| / |AB|, де × - векторний добуток

Це дозволяє визначити, наскільки далеко суглоб відхиляється від еталонної лінії.
```

**3. Моделювання площини тіла за допомогою рівняння площини:**
```
Якщо три суглоби A(x₁, y₁, z₁), B(x₂, y₂, z₂), C(x₃, y₃, z₃) мають лежати в одній площині,
то рівняння цієї площини можна записати як:
a(x-x₁) + b(y-y₁) + c(z-z₁) = 0

де (a, b, c) - нормальний вектор, що обчислюється як AB × AC.

Це дозволяє аналізувати, чи тулуб або інша частина тіла знаходиться в правильній площині.
```

**4. Визначення відстані від суглоба до площини:**
```
Якщо площина задана рівнянням ax + by + cz + d = 0, то відстань від точки P(x₀, y₀, z₀) до площини:
dist = |ax₀ + by₀ + cz₀ + d| / √(a² + b² + c²)

Це дозволяє визначити, наскільки далеко суглоб відхиляється від еталонної площини пози.
```

**5. Визначення кута між двома лініями суглобів:**
```
Якщо дві лінії утворені суглобами A₁B₁ і A₂B₂ з напрямними векторами u₁ і u₂ відповідно,
то кут між ними обчислюється як:
cos(θ) = |u₁ · u₂| / (|u₁| × |u₂|)

Це дозволяє точно вимірювати кути між частинами тіла, що важливо для аналізу йога-поз.
```

**6. Перевірка паралельності/перпендикулярності частин тіла:**
```
Дві лінії (наприклад, дві руки) паралельні, якщо їх напрямні вектори u₁ і u₂ пропорційні:
u₁ = k × u₂ для деякого скаляра k

Дві лінії перпендикулярні, якщо u₁ · u₂ = 0

Це дозволяє аналізувати симетрію та правильне розташування частин тіла в позі.
```

**7. Визначення точки перетину прямої та площини:**
```
Якщо пряма задана параметрично:
x = x₀ + ta
y = y₀ + tb
z = z₀ + tc

і площина задана рівнянням Ax + By + Cz + D = 0,
то точка перетину знаходиться підстановкою параметричних рівнянь у рівняння площини:
A(x₀ + ta) + B(y₀ + tb) + C(z₀ + tc) + D = 0

Розв'язуючи відносно t:
t = -(Ax₀ + By₀ + Cz₀ + D) / (Aa + Bb + Cc)

Підставляючи t назад, отримуємо координати точки перетину.
Це може бути корисним для аналізу, де частина тіла (наприклад, рука) має торкатися еталонної площини.
```

**8. Використання нормального рівняння прямої для аналізу відстані до еталонної пози:**
```
Нормальне рівняння прямої x × cos α + y × sin α = p дозволяє визначити відстань
від початку координат до прямої. У контексті системи, це може бути використано
для визначення відстані від центру тіла до еталонної лінії пози.
```

**9. Використання рівняння прямої у відрізках для аналізу амплітуди руху:**
```
Рівняння прямої у відрізках x/a + y/b = 1 може бути використане для визначення
граничних точок руху суглобів по осях X та Y, що важливо для аналізу амплітуди
рухів у певних йога-позах.
```

**10. Визначення рівняння прямої через дві точки для аналізу ліній суглобів:**
```
Якщо суглоби A(x₁, y₁, z₁) і B(x₂, y₂, z₂) визначають лінію тіла (наприклад, лінію плечей),
то рівняння прямої, що проходить через ці точки:
(x-x₁)/(x₂-x₁) = (y-y₁)/(y₂-y₁) = (z-z₁)/(z₂-z₁)

Це дозволяє аналізувати нахил та орієнтацію частин тіла в просторі.
```

**11. Використання загального рівняння прямої для аналізу пози:**
```
Загальне рівняння прямої Ax + By + C = 0 дозволяє визначити, чи належить суглоб
певній лінії пози. Якщо координати суглоба задовольняють рівняння, то він лежить на прямій.
Це корисно для перевірки, чи суглоби розташовані правильно відносно еталонної лінії.
```

**12. Визначення умови перетину двох прямих для аналізу симетрії:**
```
Дві прямі A₁x + B₁y + C₁ = 0 і A₂x + B₂y + C₂ = 0 перетинаються, якщо:
A₁/A₂ ≠ B₁/B₂ (умова перетину)

Це може бути використано для аналізу симетрії тіла, порівнюючи лінії лівої та правої сторін тіла.
```

Ці математичні моделі дозволяють системі точно аналізувати лінійні та площинні аспекти пози тіла, визначати відхилення від еталонної пози та надавати точні рекомендації щодо корекції рухів для досягнення правильної йога-пози.
