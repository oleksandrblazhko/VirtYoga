# Сценарії використання математичного опису для IT-проєкту "WebcamPhysExercise"

## Лекція 11. «Границя функції. Нескінченно малі та нескінченно великі функції. Визначення та основні теореми. Зв’язок між функцією, її границею та нескінченно малою»

### Обгрунтування користі:
Границі функцій є фундаментальним інструментом для аналізу безперервності рухів, збіжності до еталонної пози, стабільності параметрів та аналізу поведінки системи в критичних точках. Нескінченно малі функції дозволяють аналізувати мінімальні відхилення від ідеальної пози, що особливо важливо для точного аналізу виконання йога-вправ. Це особливо важливо для аналізу плавних переходів між позами, оцінки стабільності та виявлення аномалій у виконанні вправ.

### Зв'язок математичної формалізації із описом IT-проєкту:
- Границі функцій використовуються для аналізу збіжності до еталонної пози
- Нескінченно малі функції застосовуються для аналізу малих відхилень від пози
- Границі застосовуються для аналізу стабільності та безперервності рухів
- Зв'язок між функцією, її границею та нескінченно малою використовується для аналізу точності виконання

### Приклади математичного опису IT-проєкту:

**1. Границя функції відхилення від еталонної пози:**
```
Нехай f(t) = ||P(t) - P₀|| - функція відхилення поточної пози P(t) від еталонної P₀.
lim(t→∞) f(t) = 0 означає, що користувач з часом досягає еталонної пози.

Якщо lim(t→t₀) f(t) = L, то в момент часу t₀ відхилення наближається до значення L.
```

**2. Аналіз стабільності пози за допомогою границь:**
```
Функція варіації пози: σ(t) = √[(1/n)∑(dᵢ(t) - d̄(t))²], де dᵢ(t) - відстань i-го суглоба від середнього.

Якщо lim(t→∞) σ(t) = 0, то поза стає все більш стабільною з часом.
Якщо lim(t→∞) σ(t) = const > 0, то досягнуто стабільного рівня варіації.
```

**3. Використання нескінченно малих для аналізу малих відхилень:**
```
Нехай α(t) = ||P(t) - P₀|| - функція, що прямує до 0 при t → t₀.
Це нескінченно мала функція, що описує мале відхилення від еталонної пози.

Якщо α(t) ~ β(t) при t → t₀, то відхилення мають однаковий порядок малості.
```

**4. Зв'язок між функцією, її границею та нескінченно малою:**
```
Якщо lim(t→t₀) f(t) = A, то f(t) = A + α(t), де α(t) - нескінченно мала при t → t₀.

Це означає, що f(t) = A + α(t), де α(t) → 0 при t → t₀.
Це дозволяє аналізувати відхилення від граничного значення.
```

**5. Аналіз безперервності переходів між позами:**
```
Нехай g(t) - функція, що описує параметри пози в момент t.
Якщо lim(t→t₀⁻) g(t) = lim(t→t₀⁺) g(t) = g(t₀), то перехід у момент t₀ безперервний.

Це важливо для аналізу плавності переходів між йога-позами.
```

**6. Виявлення аномалій у виконанні:**
```
Якщо lim(t→t₀) h(t) = ∞, де h(t) - функція відхилення або помилки,
то в момент t₀ відбувається аномалія (наприклад, раптовий рух або втрата балансу).

Це може використовуватися для виявлення помилок у виконанні вправ.
```

**7. Аналіз швидкості збіжності до пози:**
```
Нехай e(t) = ||P(t) - P₀|| - помилка відстеження еталонної пози.
Якщо lim(t→∞) e(t)/t^(-k) = C ≠ 0, то збіжність має порядок k.

Це дозволяє оцінити, наскільки швидко користувач досягає правильної пози.
```

**8. Використання границь для фільтрації даних:**
```
При обробці даних від камери можна використовувати границі для згладжування:
f'(t) = lim(h→0) [f(t+h) - f(t)]/h - похідна, що дає швидкість зміни пози.

Це допомагає виявити різкі рухи та стабілізувати вимірювання.
```

**9. Аналіз точності виконання пози:**
```
Функція точності виконання: acc(t) = 1 - ||P(t) - P₀||/max_deviation

Якщо lim(t→∞) acc(t) = 1, то користувач навчився точно виконувати позу.
```

**10. Визначення стабільного стану:**
```
Якщо lim(t→∞) ||P(t) - P(t-Δt)|| = 0 для деякого Δt, то поза досягла стабільного стану.
Це означає, що поза не змінюється з часом, що важливо для утримання йога-поз.
```

**11. Аналіз відносної зміни пози:**
```
Відносна зміна пози: rel_change(t) = ||P(t) - P(t-1)|| / ||P(t-1)||

Якщо ця функція є нескінченно малою, то зміни пози є мінімальними,
що вказує на стабільне утримання пози.
```

**12. Використання границь для визначення граничних можливостей:**
```
Якщо f(t) - функція амплітуди руху, то lim(t→∞) f(t) = A_max може вказувати
на максимальну гнучкість користувача, досягнуту в результаті тренувань.
```

Ці математичні моделі дозволяють системі аналізувати безперервність рухів, збіжність до еталонної пози, стабільність параметрів, виявляти аномалії та оцінювати якість виконання йога-вправ.
