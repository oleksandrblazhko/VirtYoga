# Сценарії використання математичного опису для IT-проєкту "WebcamPhysExercise"

## Лекція 12. «Основні теореми про границі. Ознаки існування границь. Перша важлива границя. Друга важлива границя»

### Обгрунтування користі:
Основні теореми про границі дозволяють аналізувати поведінку параметрів пози тіла при різних умовах. Перша та друга важливі границі є фундаментальними інструментами для аналізу безперервності рухів, збіжності до еталонної пози, стабільності параметрів та аналізу поведінки системи в критичних точках. Це особливо важливо для аналізу плавних переходів між позами, оцінки стабільності та виявлення аномалій у виконанні вправ.

### Зв'язок математичної формалізації із описом IT-проєкту:
- Основні теореми про границі використовуються для аналізу збіжності до еталонної пози
- Перша важлива границя застосовується для нормалізації кутових параметрів
- Друга важлива границя використовується для моделювання експоненційного прогресу
- Ознаки існування границь застосовуються для аналізу стабільності та безперервності рухів

### Приклади математичного опису IT-проєкту:

**1. Застосування теореми про границю суми:**
```
Нехай f(t) = ||P_arm(t) - P_arm_ref|| і g(t) = ||P_leg(t) - P_leg_ref|| - функції відхилення
руки та ноги від еталонної пози відповідно.

Тоді lim(t→∞) [f(t) + g(t)] = lim(t→∞) f(t) + lim(t→∞) g(t)
Це дозволяє аналізувати загальну похибку як суму похибок окремих частин тіла.
```

**2. Застосування теореми про границю добутку:**
```
Якщо f(t) - функція точності виконання пози, g(t) - функція тривалості утримання пози,
то lim(t→∞) [f(t) × g(t)] = lim(t→∞) f(t) × lim(t→∞) g(t)

Це дозволяє аналізувати комплексну оцінку, що поєднує точність та тривалість.
```

**3. Перша важлива границя для нормалізації кутів:**
```
При аналізі кутів між суглобами часто використовується:
lim(θ→0) sin(θ)/θ = 1

Це дозволяє апроксимувати sin(θ) ≈ θ для малих кутів, що спрощує обчислення
при аналізі малих відхилень від еталонної пози.
```

**4. Друга важлива границя для моделювання навчання:**
```
lim(n→∞) (1 + 1/n)ⁿ = e

Ця границя використовується для моделювання експоненційного прогресу:
f(t) = A·e^(kt) або f(t) = A(1 - e^(-kt))
де A - максимальний прогрес, k - коефіцієнт навчання.
```

**5. Застосування теореми про границю складеної функції:**
```
Якщо f(t) = ||P(t) - P₀|| - відхилення від еталонної пози, а g(x) = x²,
то lim(t→∞) g(f(t)) = g(lim(t→∞) f(t))

Це дозволяє аналізувати квадратичну похибку, що часто використовується в оптимізації.
```

**6. Використання теореми про двох поліцейських:**
```
Якщо 0 ≤ error(t) ≤ stability_bound(t) і lim(t→∞) stability_bound(t) = 0,
то lim(t→∞) error(t) = 0.

Це дозволяє довести, що похибка зменшується до нуля, якщо вона обмежена
спадаючою функцією стабільності.
```

**7. Ознака існування границі для аналізу стабільності:**
```
Якщо послідовність {sₙ} оцінок стабільності пози обмежена та монотонна,
то існує lim(n→∞) sₙ = S*, що означає досягнення стабільного рівня виконання.
```

**8. Застосування другої важливої границі для аналізу прогресу:**
```
lim(x→0) (1+x)^(1/x) = e

Це може використовуватися для моделювання швидкості навчання:
Якщо прогрес вимірюється як (1 + improvement_rate)^(1/time_unit),
то при дуже малих одиницях часу це прямує до e.
```

**9. Використання границь для аналізу швидкості зміни пози:**
```
Швидкість зміни пози: lim(Δt→0) ||P(t+Δt) - P(t)||/Δt

Це дає похідну, що показує, наскільки швидко змінюється поза,
що важливо для аналізу плавності рухів.
```

**10. Аналіз відносної похибки за допомогою границь:**
```
lim(t→∞) ||P_current(t) - P_reference|| / ||P_reference||

Це дає відносну похибку, що нормалізована відносно еталонної пози,
що дозволяє порівнювати точність виконання різних поз.
```

**11. Застосування теореми про границю частки:**
```
Якщо f(t) - кількість правильних виконань, g(t) - загальна кількість спроб,
то lim(t→∞) f(t)/g(t) дає асимптотичну точність виконання пози.
```

**12. Використання важливих границь для фільтрації даних:**
```
При обробці даних про позу важливо знати, що:
lim(h→0) [sin(h)/h] = 1, що дозволяє лінеаризувати малий кутовий рух
lim(n→∞) [1 + x/n]^n = e^x, що дозволяє моделювати експоненційні зміни
```

**13. Аналіз стійкості до збурень:**
```
Якщо δ(t) - мала зміна вхідних даних (шум камери), то
lim(δ→0) ||P(t, input + δ) - P(t, input)|| = 0 вказує на стійкість системи.
```

**14. Використання границь для аналізу енергії руху:**
```
Енергія руху може бути визначена як ||velocity(t)||².
lim(t→∞) ||velocity(t)||² = 0 вказує на досягнення статичної пози.
```

Ці математичні моделі дозволяють системі аналізувати безперервність рухів, збіжність до еталонної пози, стабільність параметрів, використовувати важливі границі для нормалізації параметрів та оцінювати якість виконання йога-вправ.
