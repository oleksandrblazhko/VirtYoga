# Сценарії використання математичного опису для IT-проєкту "WebcamPhysExercise"

## Лекція 13. «Еквівалентні нескінченно малі функції. Порівняння нескінченно малих. Основні теореми про нескінченно малі функції. Застосування нескінченно малих функцій»

### Обгрунтування користі:
Еквівалентні нескінченно малі функції є важливим інструментом для аналізу малих відхилень від еталонної пози, що особливо важливо для точного аналізу виконання йога-вправ. Порівняння нескінченно малих дозволяє визначати, які відхилення є більш значущими, а які менш. Це особливо важливо для аналізу тонких різниць у виконанні поз, оцінки якості виконання та виявлення малих, але важливих відхилень від правильної техніки.

### Зв'язок математичної формалізації із описом IT-проєкту:
- Еквівалентні нескінченно малі функції використовуються для аналізу малих відхилень від пози
- Порівняння нескінченно малих застосовується для визначення значущості відхилень
- Основні теореми про нескінченно малі функції використовуються для аналізу точності виконання
- Застосування нескінченно малих функцій дозволяє аналізувати тонкі аспекти виконання вправ

### Приклади математичного опису IT-проєкту:

**1. Еквівалентність малих відхилень:**
```
Нехай α(t) = ||P(t) - P₀|| - загальна похибка від еталонної пози,
β(t) = ||J_arm(t) - J_arm_ref|| - похибка положення руки.

Якщо lim(t→t₀) α(t)/β(t) = C ≠ 0, то α(t) ~ C·β(t) при t → t₀.
Це означає, що загальна похибка пропорційна похибці руки в околі моменту t₀.
```

**2. Порівняння значущості відхилень:**
```
Якщо α(t) - відхилення тулуба, β(t) - відхилення ноги,
і lim(t→t₀) α(t)/β(t) = 0, то α(t) = o(β(t)),
що означає, що відхилення тулуба менш значуще за відхилення ноги
в аналізі загальної похибки пози.
```

**3. Використання еквівалентних нескінченно малих для спрощення:**
```
При аналізі малих кутових відхилень:
sin(θ) ~ θ, tan(θ) ~ θ, 1-cos(θ) ~ θ²/2 при θ → 0

Це дозволяє спростити аналіз кутів між суглобами при малих відхиленнях.
```

**4. Аналіз швидкості збіжності до пози:**
```
Якщо e₁(t) і e₂(t) - дві функції похибки,
і e₁(t) = o(e₂(t)) при t → ∞, то e₁(t) зменшується швидше за e₂(t).

Це дозволяє порівнювати ефективність різних аспектів виконання пози.
```

**5. Визначення порядку малості відхилень:**
```
Якщо ||P(t) - P₀|| ~ C·t^k при t → 0, то відхилення має порядок k.

Це дозволяє аналізувати, наскільки швидко користувач досягає правильної пози.
```

**6. Застосування нескінченно малих для аналізу стабільності:**
```
Якщо σ(t) - функція варіації пози, і σ(t) → 0 при t → ∞,
то σ(t) є нескінченно малою, що вказує на зростання стабільності з часом.
```

**7. Порівняння ефективності корекцій:**
```
Нехай c₁(t) і c₂(t) - функції покращення після двох різних корекцій.
Якщо c₁(t) ~ k·c₂(t), то корекції мають однаковий ефект з коефіцієнтом k.
```

**8. Використання ln(1+x) ~ x для аналізу відносного прогресу:**
```
Якщо s(t) - оцінка виконання пози, то відносне покращення:
ln(s(t+Δt)/s(t)) ~ (s(t+Δt) - s(t))/s(t) при малих змінах.

Це дозволяє аналізувати відносний прогрес більш точно.
```

**9. Аналіз експоненційного наближення до пози:**
```
e^x - 1 ~ x при x → 0, тому відхилення вигляду e^(error) - 1 ~ error.

Це корисно для аналізу експоненційного наближення до еталонної пози.
```

**10. Використання √(1+x) - 1 ~ x/2 для аналізу норм:**
```
При аналізі норм векторів: √(1+δ) - 1 ~ δ/2 при δ → 0.

Це дозволяє спростити аналіз малих змін у нормалізованих векторах суглобів.
```

**11. Порівняння швидкості зменшення похибок:**
```
Якщо e_arm(t) і e_leg(t) - похибки руки та ноги відповідно,
і lim(t→∞) e_arm(t)/e_leg(t) = 0, то рука досягає еталонної пози швидше за ногу.
```

**12. Аналіз відносної важливості суглобів:**
```
Якщо αᵢ(t) - відхилення i-го суглоба, і αᵢ(t) = o(αⱼ(t)) при t → t₀,
то відхилення j-го суглоба більш критичне для загальної похибки в момент t₀.
```

**13. Застосування нескінченно малих для фільтрації шуму:**
```
При обробці даних від камери, якщо шум δ(t) є нескінченно малим,
то його вплив на аналіз пози можна оцінити як o(1) при t → t₀.
```

**14. Визначення домінуючих факторів похибки:**
```
Якщо загальна похибка E(t) = α₁(t) + α₂(t) + ... + αₙ(t),
і α₁(t) домінує над іншими (αᵢ(t) = o(α₁(t)) для i ≠ 1),
то α₁(t) є ключовим фактором, який потрібно виправляти.
```

**15. Аналіз ефективності тренувань за допомогою еквівалентних функцій:**
```
Якщо прогрес описується функціями f(t) і g(t), і f(t) ~ g(t) при t → ∞,
то обидві функції мають однакову асимптотичну ефективність.
```

**16. Використання асимптотичних розкладів для точного аналізу:**
```
sin(θ) = θ - θ³/6 + o(θ⁴) дозволяє більш точно аналізувати кутові відхилення
при необхідності високої точності аналізу пози.
```

Ці математичні моделі дозволяють системі аналізувати тонкі відмінності в виконанні поз, визначати найбільш критичні відхилення, порівнювати ефективність різних аспектів виконання та забезпечувати точний аналіз малих відхилень від еталонної моделі йога-пози.
