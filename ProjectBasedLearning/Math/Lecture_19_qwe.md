## Обгрунтування користі від використання змісту лекції та практичного заняття для успішного виконання IT-проєкту:

У системі контролю рухів людини під час виконання йога-вправ, методи дослідження функцій за допомогою похідних мають критичне значення для аналізу та оптимізації рухових параметрів. Похідні дозволяють визначати швидкість зміни положення суглобів, що важливо для аналізу плавності рухів та виявлення екстремальних значень параметрів, які можуть свідчити про неправильне виконання вправи.

Правило Лопіталя особливо корисне при аналізі невизначених форм, що можуть виникати при обчисленні границь функцій, що описують рухи. Теореми Лагранжа та Ролля застосовуються для аналізу поведінки функцій, що моделюють рухи тіла, і визначення точок, де похідна дорівнює нулю, що відповідає стаціонарним станам або екстремальним позиціям.

Дослідження функцій на монотонність та екстремуми дозволяє системі визначати, коли рух досягає максимальних або мінімальних значень, що важливо для контролю правильного виконання пози. Визначення опуклості та точок перегину допомагає аналізувати зміну характеру руху (наприклад, з прискорення на уповільнення).

Практичне заняття навчає студентів застосовувати ці методи на практиці, що необхідно для аналізу реальних даних, що надходять від системи відстеження рухів.

## Зв'язок математичної формалізації із описом IT-проєкту:

У системі розпізнавання рухів, кожен суглоб має координати (x, y, z), які змінюються в часі. Ці координати можна розглядати як функції часу: x(t), y(t), z(t). Похідні цих функцій x'(t), y'(t), z'(t) представляють швидкість зміни координат суглоба в кожен момент часу.

- Похідна x'(t) = dx/dt - швидкість зміни x-координати суглоба
- Похідна y'(t) = dy/dt - швидкість зміни y-координати суглоба  
- Похідна z'(t) = dz/dt - швидкість зміни z-координати суглоба

Другі похідні x''(t), y''(t), z''(t) представляють прискорення суглоба.

Аналіз цих функцій за допомогою похідних дозволяє:
- Визначити монотонність руху (зростання або спадання координат)
- Знайти екстремальні точки (максимуми та мінімуми), що відповідають критичним позиціям
- Визначити опуклість траєкторії руху та точки перегину
- Застосувати правило Лопіталя для аналізу невизначених форм при обчисленні границь
- Використовувати формулу Тейлора для апроксимації складних рухових функцій

Практичне заняття навчає студентів застосовувати ці методи для реальних функцій, що моделюють рухи тіла, що необхідно для аналізу даних системи відстеження.

## Приклади математичного опису IT-проєкту:

### Приклад 1: Застосування теореми Лагранжа для аналізу руху суглоба
Нехай функція x(t) описує зміну x-координати плечового суглоба на інтервалі [t₁, t₂]. За теоремою Лагранжа, існує точка τ ∈ (t₁, t₂), така що:
x(t₂) - x(t₁) = x'(τ)(t₂ - t₁)

Це означає, що середня швидкість зміни координати на інтервалі [t₁, t₂] дорівнює миттєвій швидкості в деякий момент τ. Це дозволяє системі аналізувати, чи рух відбувався з постійною швидкістю чи з прискоренням.

### Приклад 2: Дослідження функції на монотонність для аналізу руху
Якщо x'(t) > 0 на інтервалі (a, b), то рух суглоба за координатою x є монотонно зростаючим на цьому інтервалі. Якщо x'(t) < 0, то рух є монотонно спадаючим. Це важливо для аналізу направленості руху під час виконання йога-вправи.

### Приклад 3: Визначення екстремумів для знаходження критичних позицій
Якщо x'(t₀) = 0 і x''(t₀) < 0, то в момент t₀ функція x(t) має локальний максимум - це може відповідати максимальному відхиленню суглоба в певному напрямку. Якщо x''(t₀) > 0, то це локальний мінімум - мінімальне положення суглоба.

### Приклад 4: Застосування правила Лопіталя для аналізу невизначених форм
При аналізі відношення двох функцій, що описують рух, може виникнути невизначеність типу 0/0:
lim(t→t₀) [f(t)/g(t)] = lim(t→t₀) [f'(t)/g'(t)], якщо f(t₀) = g(t₀) = 0

Це може бути корисно при аналізі відношення швидкостей руху двох суглобів в момент, коли обидва мають нульову швидкість.

### Приклад 5: Використання формули Тейлора для апроксимації руху
Формула Тейлора дозволяє апроксимувати складну функцію руху суглоба:
x(t) ≈ x(t₀) + x'(t₀)(t - t₀) + x''(t₀)(t - t₀)²/2! + ...

Це дозволяє системі прогнозувати майбутнє положення суглоба на основі поточного положення, швидкості та прискорення, що важливо для передбачення руху та вчасного надання рекомендацій.

### Приклад 6: Застосування практичних навичок з практичного заняття
На практичному занятті студенти навчаються знаходити найбільше і найменше значення функції на відрізку. Це застосовується до аналізу рухів, наприклад, для визначення максимальної амплітуди руху суглоба під час виконання вправи:
Для функції x(t) на відрізку [a, b]:
1. Знаходимо критичні точки: x'(t) = 0
2. Обчислюємо значення функції в критичних точках та на кінцях відрізка
3. Вибираємо найбільше і найменше значення

Це дозволяє визначити межі руху суглоба під час виконання певної йога-вправи.

### Приклад 7: Аналіз опуклості та точок перегину траєкторії руху
Якщо x''(t) > 0, то траєкторія руху є опуклою вгору (прискорення додатне). Якщо x''(t) < 0, то траєкторія опукла вниз (прискорення від'ємне). Точка перегину, де x''(t) = 0, відповідає моменту зміни характеру руху (з прискорення на уповільнення або навпаки).

Це дозволяє системі аналізувати динаміку виконання вправи та визначати моменти, коли користувач змінює характер руху.
