# Лекція 20: Інтеграл для відновлення траєкторії руху

### Обґрунтування користі для IT-проєкту

Інтегрування є операцією, оберненою до диференціювання. Якщо похідна дозволяє знайти швидкість за відомою траєкторією, то **інтеграл (первісна)** дозволяє **відновити траєкторію руху за відомою швидкістю**. Це відкриває можливості для планування та аналізу динамічних вправ.
-   **Планування траєкторії:** Програма може визначити ідеальну *функцію швидкості* `v(t)` для плавного руху (наприклад, швидкість, що плавно зростає, а потім плавно спадає). Проінтегрувавши цю функцію, програма отримує ідеальну *функцію положення* `p(t) = ∫v(t)dt`, з якою потім порівнює рух користувача.
-   **Обчислення сумарної зміни:** Інтеграл від швидкості зміни деякої величини дає загальну зміну цієї величини. Наприклад, проінтегрувавши кутову швидкість згинання ліктя `ω(t)` за час `Δt`, програма отримає загальний кут, на який зігнувся лікоть.
-   **Методи інтегрування:** Такі техніки, як **заміна змінної** та **інтегрування частинами**, є необхідними інструментами для розробника, щоб вирішувати інтеграли, що виникають при моделюванні складних рухів (наприклад, рух з коливальною швидкістю).

### Зв'язок математичної формалізації із описом IT-проєкту

| Елемент опису IT-проєкту | Математична формалізація (з розділів 11.1, 11.2) |
| :--- | :--- |
| Відновлення положення суглоба `p(t)` з його швидкості `v(t)` | **Невизначений інтеграл**: `p(t) = ∫v(t)dt`. `p(t)` є **первісною** для `v(t)`. |
| Загальний кут повороту за час `T` | Визначений інтеграл (буде розглянуто пізніше), але ідея та ж: `Δθ = ∫ω(t)dt` від 0 до `T`. |
| Моделювання руху з коливальною швидкістю (напр., `v(t)=t*cos(t)`) | Розв'язання інтеграла `∫t*cos(t)dt` за допомогою **інтегрування частинами**. |
| Моделювання руху, де швидкість залежить від положення | Розв'язання інтеграла за допомогою **заміни змінної**. |

### Приклади математичного опису IT-проєкту

**1. Побудова ідеальної траєкторії для динамічної вправи:**

-   **Вправа:** "Плавно підніміть руку перед собою за 2 секунди".
-   **Моделювання швидкості:** Програма хоче, щоб рух був найплавнішим на початку і в кінці. Вона може задати ідеальну функцію кутової швидкості `ω(t)`, наприклад, `ω(t) = k * sin(π*t/2)` (швидкість максимальна в середині руху і нульова на початку та в кінці).
-   **Знаходження траєкторії:** Щоб знати, де має бути рука в будь-який момент часу `t`, програма має знайти функцію кута `θ(t)`. Для цього вона бере інтеграл:
    `θ(t) = ∫ω(t)dt = ∫k*sin(πt/2)dt`.
-   Використовуючи таблицю інтегралів та заміну змінної (`u = πt/2`), програма знаходить:
    `θ(t) = - (2k/π) * cos(πt/2) + C`.
-   **Результат:** Тепер у програми є точна формула `θ(t)`, яка описує ідеальне положення руки в будь-який момент часу. Вона може порівнювати реальний кут з цим ідеальним значенням і давати точні підказки.

**2. Аналіз складного руху (Інтегрування частинами):**

-   **Вправа:** Виконати рух, де швидкість руки лінійно зростає, але при цьому коливається (наприклад, для тренування координації).
-   **Модель швидкості:** `v(t) = t * cos(t)`.
-   **Знаходження траєкторії:** Щоб зрозуміти, яку траєкторію `p(t)` має описувати рука, програма повинна обчислити `p(t) = ∫t*cos(t)dt`.
-   **Розв'язання:** Цей інтеграл належить до стандартного типу, що розв'язується **інтегруванням частинами** (розділ 11.2).
    -   `u = t`, `dv = cos(t)dt`
    -   `du = dt`, `v = sin(t)`
    -   `∫t*cos(t)dt = t*sin(t) - ∫sin(t)dt = t*sin(t) + cos(t) + C`.
-   **Результат:** Програма отримала аналітичну формулу для ідеальної траєкторії і може використовувати її для порівняння з рухами користувача.

**3. Зв'язок між кутом та швидкістю (Заміна змінної):**

-   Припустимо, програма аналізує рух, де швидкість `v` залежить не від часу, а від кута `θ` в суглобі: `v(θ) = f(θ)`. Але нам потрібна залежність `θ(t)`.
-   Ми знаємо, що `v = dθ/dt`. Отже, `dt = dθ/v = dθ/f(θ)`.
-   Проінтегрувавши обидві частини, отримаємо: `t = ∫ dθ / f(θ)`.
-   **Результат:** Розв'язавши цей інтеграл (можливо, методом **заміни змінної**), програма знаходить залежність часу від кута `t(θ)`, а обернувши її — шукану траєкторію `θ(t)`.
