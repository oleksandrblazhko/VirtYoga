## Обгрунтування користі від використання змісту лекції та практичного заняття для успішного виконання IT-проєкту:

У системі контролю рухів людини під час виконання йога-вправ, поняття первісної та невизначеного інтеграла мають важливе значення для аналізу даних про рух. Якщо похідна дозволяє визначити швидкість зміни положення суглобів, то інтеграл дозволяє відновити функцію положення за відомою функцією швидкості, що важливо для аналізу та прогнозування рухів.

Коли система відстеження рухів надає дані про швидкість зміни координат суглобів, інтегрування дозволяє відновити абсолютні значення координат, що необхідно для аналізу повної траєкторії руху. Це особливо корисно при обробці даних, коли відбувається втрата окремих кадрів або шум у даних.

Практичне заняття, присвячене табличним інтегралам, заміні змінної та інтегруванню частинами, має безпосереднє застосування до аналізу складних функцій, що описують рухи тіла. На практиці студенти навчаються обчислювати інтеграли для функцій, що моделюють рухи тіла, що необхідно для аналізу даних системи відстеження.

## Зв'язок математичної формалізації із описом IT-проєкту:

У системі розпізнавання рухів, якщо відома функція швидкості руху суглоба v(t) = (vx(t), vy(t), vz(t)), то для відновлення функції положення S(t) = (x(t), y(t), z(t)) необхідно знайти первісні:

x(t) = ∫ vx(t)dt + C₁
y(t) = ∫ vy(t)dt + C₂
z(t) = ∫ vz(t)dt + C₃

де C₁, C₂, C₃ - сталі інтегрування, які визначаються з початкових умов.

Практичне заняття навчає студентів методам інтегрування, що необхідно для обробки реальних даних, які можуть бути представлені у вигляді складних функцій, що потребують застосування методів заміни змінної або інтегрування частинами.

Наприклад, при аналізі прискорення руху, якщо відома функція прискорення a(t), то швидкість визначається як:
v(t) = ∫ a(t)dt + C

Це дозволяє відновлювати функції руху за даними датчиків, що вимірюють прискорення.

## Приклади математичного опису IT-проєкту:

### Приклад 1: Відновлення функції положення за відомою швидкістю
Нехай відома функція швидкості плечового суглоба:
v(t) = (vx(t), vy(t), vz(t)) = (2t + 1, cos(t), e^t)

Для відновлення функції положення S(t) = (x(t), y(t), z(t)) знаходимо первісні:
x(t) = ∫ (2t + 1)dt = t² + t + C₁
y(t) = ∫ cos(t)dt = sin(t) + C₂
z(t) = ∫ e^t dt = e^t + C₃

де сталі C₁, C₂, C₃ визначаються з початкових умов (початкового положення суглоба).

### Приклад 2: Застосування методу заміни змінної (практичне заняття)
При аналізі руху, що описується функцією:
∫ sin(3t + 5)dt

Застосовуємо метод заміни змінної:
u = 3t + 5, du = 3dt, dt = du/3
∫ sin(u) × (du/3) = (1/3)∫ sin(u)du = -(1/3)cos(u) + C = -(1/3)cos(3t + 5) + C

Це дозволяє обробляти сигнали, що містять тригонометричні функції з лінійною підстановкою, що часто трапляється при аналізі періодичних рухів.

### Приклад 3: Інтегрування частинами (практичне заняття)
Для аналізу руху, що описується добутком функцій, наприклад:
∫ t × cos(t)dt

Застосовуємо метод інтегрування частинами:
∫ udv = uv - ∫ vdu
де u = t, dv = cos(t)dt
тоді du = dt, v = sin(t)

∫ t × cos(t)dt = t × sin(t) - ∫ sin(t)dt = t × sin(t) + cos(t) + C

Це корисно при аналізі рухів, де амплітуда змінюється з часом.

### Приклад 4: Використання таблиці інтегралів (практичне заняття)
При обробці даних системи відстеження часто зустрічаються стандартні функції, для яких використовуються табличні інтеграли:
- ∫ e^x dx = e^x + C
- ∫ sin(x) dx = -cos(x) + C
- ∫ cos(x) dx = sin(x) + C
- ∫ 1/x dx = ln|x| + C

Ці інтеграли необхідні для аналізу експоненційних, тригонометричних та логарифмічних моделей руху.

### Приклад 5: Інтегрування функцій, що містять квадратний тричлен
При аналізі рухів, що мають параболічну траєкторію:
∫ 1/(t² + 4t + 5) dt

Виділяємо повний квадрат:
t² + 4t + 5 = (t + 2)² + 1

∫ 1/((t + 2)² + 1) dt = arctan(t + 2) + C

Це дозволяє аналізувати рухи, що описуються функціями з квадратичними виразами в знаменнику.

### Приклад 6: Відновлення траєкторії руху за даними прискорення
Якщо датчики надають дані про прискорення руху суглоба:
a(t) = (2, -sin(t), 3e^(2t))

Для відновлення швидкості:
v(t) = ∫ a(t)dt = (∫ 2dt, ∫ -sin(t)dt, ∫ 3e^(2t)dt) = (2t + C₁, cos(t) + C₂, (3/2)e^(2t) + C₃)

Для відновлення положення:
S(t) = ∫ v(t)dt = (t² + C₁t + D₁, sin(t) + C₂t + D₂, (3/4)e^(2t) + C₃t + D₃)

Це дозволяє відновлювати повну траєкторію руху за даними про прискорення, що важливо при обробці даних з акселерометрів.

Ці математичні поняття дозволяють системі аналізувати та відновлювати траєкторії руху суглобів, що важливо для оцінки якості виконання йога-вправ та виявлення відхилень від еталонної пози.
