### Обґрунтування користі для IT-проєкту

Ця лекція надає "спеціалізовані інструменти" для розв'язання складних інтегралів, які неминуче виникають при точному математичному моделюванні біомеханіки людського тіла.
-   **Інтегрування тригонометричних функцій:** Кінематика будь-якого скелета (включаючи людину) описується через кути в суглобах, тому функції `sin(θ)` та `cos(θ)` є основою всіх розрахунків. Якщо ми хочемо відновити траєкторію `p(t)` зі швидкості `v(t)`, яка сама залежить від `sin(θ(t))` та `cos(θ(t))`, нам доведеться інтегрувати складні тригонометричні вирази. Ця лекція дає методи для цього.
-   **Універсальна тригонометрична підстановка:** Це потужний, хоч і громіздкий, метод, який гарантує, що будь-який інтеграл, що раціонально залежить від `sin(x)` та `cos(x)`, може бути зведений до інтегралу від раціонального дробу. Для розробника це означає, що існує гарантований алгоритм для розв'язання широкого класу кінематичних задач.
-   **Інтегрування раціональних дробів:** Це кінцевий етап, до якого зводяться багато складних інтегралів (зокрема, після універсальної тригонометричної підстановки). Вміння працювати з ними є необхідною технічною навичкою.

### Зв'язок математичної формалізації із описом IT-проєкту

| Елемент опису IT-проєкту | Математична формалізація (з розділу 11.3) |
| :--- | :--- |
| Відновлення траєкторії з швидкості, що залежить від кутів | **Інтегрування тригонометричних функцій**. `p(t) = ∫ f(sin(θ(t)), cos(θ(t))) dt`. |
| Гарантований метод розв'язання складних кінематичних рівнянь | **Універсальна тригонометрична підстановка** `u = tan(t/2)`, що зводить тригонометричний інтеграл до інтегралу від **раціонального дробу**. |
| Розрахунок енергії, що витрачається на подолання опору, пропорційного `v²` | Інтеграл може зводитись до `∫ dv / (a - bv²)`, що є **раціональним дробом** відносно `v`. |

### Приклади математичного опису IT-проєкту

**1. Моделювання руху з опором, залежним від орієнтації:**

-   **Завдання:** Користувач робить рух рукою у в'язкому середовищі (наприклад, у воді, або це може бути модель для людей з обмеженою рухливістю).
-   **Модель:** Припустимо, швидкість зміни кута `dθ/dt` обернено пропорційна опору, який, у свою чергу, залежить від орієнтації кінцівки, наприклад: `dθ/dt = 1 / (2 + cos(θ))`.
-   **Знаходження залежності θ(t):** Щоб знайти, як кут `θ` змінюється з часом `t`, потрібно розділити змінні і проінтегрувати:
    `dt = (2 + cos(θ))dθ  =>  t = ∫(2 + cos(θ))dθ`.
-   **Розв'язання:** Інтеграл `∫cos(θ)dθ` є табличним, але якби функція була складнішою, наприклад `1 / (2 + cos(θ))`, довелося б застосовувати **універсальну тригонометричну підстановку**, щоб звести його до інтегралу від раціонального дробу. Це дозволяє програмі побудувати точну модель руху для складних фізичних умов.

**2. Точний розрахунок траєкторії обертання:**

-   **Завдання:** Програма має розрахувати точну траєкторію `(x(t), y(t))` для точки, яка обертається з кутовою швидкістю `ω`, що залежить від квадрату синуса часу: `ω(t) = dθ/dt = k*sin²(t)`.
-   **Знаходження кута:** `θ(t) = ∫ k*sin²(t) dt`.
-   **Розв'язання:** Цей інтеграл не є базовим табличним. Для його взяття необхідно використати тригонометричну формулу пониження степеня: `sin²(t) = (1 - cos(2t))/2`.
    `θ(t) = (k/2) ∫(1 - cos(2t))dt = (k/2) * (t - (1/2)sin(2t)) + C`.
-   **Результат:** Програма отримує точну (а не наближену) формулу для кута `θ` в будь-який момент часу `t`. Підставивши `θ(t)` у рівняння `x(t)=R*cos(θ(t))` та `y(t)=R*sin(θ(t))`, вона отримує ідеальну траєкторію для порівняння з рухами користувача.

**3. Розкладання на прості рухи (Раціональні дроби):**

-   Після застосування універсальної тригонометричної підстановки у складному кінематичному завданні, може виникнути інтеграл від раціонального дробу, наприклад: `∫ (t+1) / (t² * (t-2)) dt`.
-   Метод розкладання на прості доданки (описаний в розділі 11.3) дозволяє переписати цей складний дріб як суму простіших: `A/t + B/t² + C/(t-2)`.
-   Кожен з цих інтегралів (`∫A/t dt`, `∫B/t² dt`, `∫C/(t-2) dt`) є простим табличним інтегралом.
-   **Результат:** Програма може розв'язати початково складну задачу, розбивши її на суму елементарних, легко обчислюваних рухів.
