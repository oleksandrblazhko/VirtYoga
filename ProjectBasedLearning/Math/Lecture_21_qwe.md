## Обгрунтування користі від використання змісту лекції та практичного заняття для успішного виконання IT-проєкту:

У системі контролю рухів людини під час виконання йога-вправ, методи інтегрування раціональних дробів, ірраціональностей та тригонометричних функцій мають важливе значення для аналізу складних рухових патернів. Багато фізичних моделей, що описують рух тіла, призводять до диференційних рівнянь, розв'язання яких потребує інтегрування функцій різного типу.

Наприклад, при аналізі руху суглобів часто виникають функції, що містять квадратні корені (ірраціональності), раціональні вирази (відношення поліномів) або тригонометричні функції. Методи інтегрування цих функцій дозволяють відновлювати траєкторії руху, обчислювати загальні параметри руху (наприклад, загальну енергію або загальний шлях) та аналізувати динаміку виконання вправ.

Практичне заняття, присвячене інтегруванню дробово-раціональних функцій, ірраціональностей та тригонометричних функцій, має безпосереднє застосування до аналізу реальних даних, що надходять від системи відстеження рухів. На практиці студенти навчаються застосовувати різні методи інтегрування до функцій, що моделюють рухи тіла, що необхідно для аналізу даних системи відстеження.

## Зв'язок математичної формалізації із описом IT-проєкту:

У системі розпізнавання рухів, при аналізі складних рухів тіла часто виникають функції, що потребують застосування різних методів інтегрування:

1. Раціональні функції - виникають при аналізі відношень між різними параметрами руху
2. Ірраціональні функції - з'являються при обчисленні відстаней між суглобами, довжин траєкторій руху
3. Тригонометричні функції - описують періодичні рухи та кути між суглобами

Практичне заняття навчає студентів застосовувати методи інтегрування до реальних функцій, що моделюють рухи тіла, що необхідно для обробки даних системи відстеження.

Наприклад, при обчисленні довжини траєкторії руху суглоба:
L = ∫ √[(dx/dt)² + (dy/dt)² + (dz/dt)²] dt

Це ірраціональний інтеграл, що вимагає застосування спеціальних методів інтегрування.

## Приклади математичного опису IT-проєкту:

### Приклад 1: Інтегрування раціональних функцій
При аналізі відношення швидкостей двох суглобів може виникнути функція:
∫ (t² + 1)/(t² - 1) dt

Розкладаємо на простіші дроби:
(t² + 1)/(t² - 1) = (t² + 1)/((t - 1)(t + 1)) = A/(t - 1) + B/(t + 1)

Після розрахунку: A = 1, B = 1
∫ (1/(t - 1) + 1/(t + 1)) dt = ln|t - 1| + ln|t + 1| + C = ln|t² - 1| + C

Це дозволяє аналізувати відношення швидкостей руху різних суглобів.

### Приклад 2: Інтегрування ірраціональностей
При обчисленні довжини траєкторії руху суглоба:
∫ √(1 + t²) dt

Застосовуємо тригонометричну підстановку: t = tan(θ), dt = sec²(θ)dθ
∫ √(1 + tan²(θ)) × sec²(θ)dθ = ∫ sec(θ) × sec²(θ)dθ = ∫ sec³(θ)dθ

Цей інтеграл обчислюється методом інтегрування частинами:
∫ sec³(θ)dθ = (1/2)[sec(θ)tan(θ) + ln|sec(θ) + tan(θ)|] + C

Повертаючись до змінної t:
∫ √(1 + t²) dt = (1/2)[t√(1 + t²) + ln|t + √(1 + t²)|] + C

Це дозволяє обчислювати загальну довжину траєкторії руху суглоба.

### Приклад 3: Інтегрування тригонометричних функцій (практичне заняття)
При аналізі періодичних рухів, що часто трапляються в йогі:
∫ sin²(t) dt

Використовуємо тригонометричну тотожність: sin²(t) = (1 - cos(2t))/2
∫ sin²(t) dt = ∫ (1 - cos(2t))/2 dt = (1/2)∫ dt - (1/2)∫ cos(2t) dt = t/2 - sin(2t)/4 + C

Це дозволяє аналізувати енергію періодичних рухів, оскільки енергія часто пропорційна квадрату синуса або косинуса.

### Приклад 4: Універсальна тригонометрична підстановка (практичне заняття)
Для інтегралів, що містять sin(t) та cos(t), застосовуємо підстановку:
u = tan(t/2), тоді sin(t) = 2u/(1 + u²), cos(t) = (1 - u²)/(1 + u²), dt = 2du/(1 + u²)

Приклад: ∫ 1/(2 + sin(t)) dt
Після підстановки:
∫ 1/(2 + 2u/(1 + u²)) × 2du/(1 + u²) = ∫ 2du/((2(1 + u²) + 2u)(1 + u²)/(1 + u²)) = ∫ 2du/(2 + 2u² + 2u) = ∫ du/(1 + u² + u)

Це дозволяє аналізувати складні тригонометричні залежності, що виникають при моделюванні рухів тіла.

### Приклад 5: Інтегрування ірраціональностей за допомогою тригонометричних підстановок (практичне заняття)
При аналізі відстаней між суглобами часто виникає:
∫ 1/√(a² - x²) dx

Застосовуємо підстановку: x = a sin(θ), dx = a cos(θ)dθ
∫ 1/√(a² - a²sin²(θ)) × a cos(θ)dθ = ∫ 1/(a cos(θ)) × a cos(θ)dθ = ∫ dθ = θ + C = arcsin(x/a) + C

Це дозволяє обчислювати кути та відстані між суглобами, що важливо для аналізу правильності виконання пози.

### Приклад 6: Застосування методів інтегрування до аналізу енергії руху
При аналізі енергії руху суглоба, якщо швидкість задана як v(t) = √(sin²(t) + cos²(t) + 1) = √2, то кінетична енергія (пропорційна v²(t)) постійна.

Але якщо v(t) = √(sin²(t) + cos²(t) + t²) = √(1 + t²), то для обчислення загальної енергії за певний проміжок часу:
∫ v²(t) dt = ∫ (1 + t²) dt = t + t³/3 + C

Це дозволяє аналізувати енергетичні характеристики рухів, що важливо для оцінки ефективності виконання вправ.

Ці математичні поняття дозволяють системі аналізувати складні рухові патерни, обчислювати енергетичні параметри рухів та визначати оптимальні траєкторії для виконання йога-вправ.
