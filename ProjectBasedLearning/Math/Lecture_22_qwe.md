## Номер лекції: 22
## Назва лекції: Визначений інтеграл. Означення та властивості. Інтеграли зі змінною верхнею межею. Теорема про диференціювання інтегралів зі змінною верхнею межею. Формула Ньютона-Лейбніца

## Назва практичного заняття: Формула Ньютона-Лейбніца. Заміна змінної і інтегрування частинами у визначеному інтегралі

## Обгрунтування користі від використання змісту лекції та практичного заняття для успішного виконання IT-проєкту:

У системі контролю рухів людини під час виконання йога-вправ, поняття визначеного інтеграла має важливе значення для аналізу кількісних характеристик рухів. Визначений інтеграл дозволяє обчислювати загальні параметри руху, такі як загальний шлях, енергія, площа під кривою активності, що важливо для кількісної оцінки якості виконання вправ.

Інтеграли зі змінною верхньою межею особливо корисні для аналізу накопичених параметрів руху протягом часу. Наприклад, якщо відома швидкість зміни кута між суглобами, то інтеграл зі змінною верхньою межею дозволяє обчислити загальну зміну кута до певного моменту часу.

Формула Ньютона-Лейбніца дозволяє ефективно обчислювати визначені інтеграли за допомогою первісних, що важливо для швидкого аналізу даних системи в реальному часі. Практичне заняття, присвячене застосуванню формули Ньютона-Лейбніца, заміні змінної та інтегруванню частинами у визначеному інтегралі, має безпосереднє застосування до аналізу складних функцій, що описують рухи тіла.

## Зв'язок математичної формалізації із описом IT-проєкту:

У системі розпізнавання рухів, при аналізі даних часто виникають задачі, що потребують обчислення визначених інтегралів:

1. Обчислення загального переміщення суглоба за певний час: ∫ v(t) dt
2. Обчислення загальної енергії руху: ∫ F(t)v(t) dt
3. Обчислення площі під кривою активності: ∫ a(t) dt
4. Обчислення накопиченого значення параметра: ∫₀^t f(τ) dτ

Практичне заняття навчає студентів застосовувати методи обчислення визначених інтегралів до реальних функцій, що моделюють рухи тіла, що необхідно для обробки даних системи відстеження.

Формула Ньютона-Лейбніца: ∫ₐᵇ f(x)dx = F(b) - F(a), де F(x) - первісна для f(x), дозволяє ефективно обчислювати визначені інтеграли, що важливо для аналізу даних у реальному часі.

## Приклади математичного опису IT-проєкту:

### Приклад 1: Обчислення загального переміщення суглоба
Нехай швидкість руху суглоба задана як v(t) = 2t + 1 (в умовних одиницях). Для обчислення загального переміщення за інтервал часу [0, 3]:

∫₀³ (2t + 1) dt = [t² + t]₀³ = (3² + 3) - (0² + 0) = 9 + 3 = 12

Це дозволяє обчислити загальну відстань, на яку перемістився суглоб за певний час.

### Приклад 2: Інтеграли зі змінною верхньою межею
Для аналізу накопиченого руху до моменту часу t, визначаємо:
S(t) = ∫₀^t (2τ + 1) dτ = [τ² + τ]₀^t = t² + t

Це дозволяє відстежувати накопичене переміщення суглоба в будь-який момент часу, що важливо для аналізу прогресу користувача.

### Приклад 3: Застосування формули Ньютона-Лейбніца (практичне заняття)
Для обчислення ∫₁² 3t² dt:
∫₁² 3t² dt = [t³]₁² = 2³ - 1³ = 8 - 1 = 7

Це дозволяє обчислювати загальну зміну параметра за певний інтервал, що важливо для аналізу ефективності виконання вправи.

### Приклад 4: Заміна змінної у визначеному інтегралі (практичне заняття)
Для обчислення ∫₀^(π/2) sin(2t) dt, застосовуємо заміну: u = 2t, du = 2dt
Коли t = 0, u = 0; коли t = π/2, u = π
∫₀^(π/2) sin(2t) dt = ∫₀^π sin(u) (du/2) = (1/2)[-cos(u)]₀^π = (1/2)[-cos(π) + cos(0)] = (1/2)[1 + 1] = 1

Це дозволяє аналізувати періодичні рухи, що часто трапляються в йогі.

### Приклад 5: Інтегрування частинами у визначеному інтегралі (практичне заняття)
Для обчислення ∫₀^π t·cos(t) dt, застосовуємо інтегрування частинами:
u = t, dv = cos(t)dt
du = dt, v = sin(t)

∫₀^π t·cos(t) dt = [t·sin(t)]₀^π - ∫₀^π sin(t) dt = [π·sin(π) - 0·sin(0)] - [-cos(t)]₀^π
= [0 - 0] - [-cos(π) + cos(0)] = 0 - [-(-1) + 1] = 0 - [1 + 1] = -2

Це дозволяє аналізувати рухи, де амплітуда змінюється з часом, що важливо для аналізу динаміки виконання вправи.

### Приклад 6: Обчислення площі під кривою активності
Якщо функція a(t) = 4sin²(t) описує рівень активності руху, то загальна активність за інтервал [0, π]:
∫₀^π 4sin²(t) dt = 4∫₀^π (1-cos(2t))/2 dt = 2∫₀^π (1-cos(2t)) dt = 2[t - sin(2t)/2]₀^π = 2[π - sin(2π)/2 - 0] = 2π

Це дозволяє кількісно оцінити загальну активність користувача під час виконання вправи.

### Приклад 7: Застосування властивостей визначеного інтеграла
Для аналізу симетричних рухів, використовуємо властивість:
∫₋ₐ^ᵃ f(x)dx = 2∫₀^ᵃ f(x)dx, якщо f(x) парна
∫₋ₐ^ᵃ f(x)dx = 0, якщо f(x) непарна

Наприклад, для ∫₋₁^¹ t·cos(t) dt: функція t·cos(t) є непарною (t непарна, cos(t) парна, добуток непарна), тому інтеграл дорівнює 0.

Це дозволяє спрощувати обчислення для симетричних рухів, що часто трапляються в йогі.

Ці математичні поняття дозволяють системі аналізувати кількісні характеристики рухів, обчислювати енергетичні параметри та визначати оптимальні траєкторії для виконання йога-вправ.
