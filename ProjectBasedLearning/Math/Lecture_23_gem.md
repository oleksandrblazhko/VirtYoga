# Лекція 23: Застосування визначеного інтеграла для оцінки якості вправ

**Назва лекції:** «Застосування визначеного інтеграла. Обчислення площі, довжини дуги, об’єму тіла... Невласні інтеграли».

**Назва практичного заняття:** «Застосування визначеного інтеграла до задач геометрії та розв’язку задач фізики та механіки».

**Пов'язаний розділ з `Docs\Lecture_1_9.txt`:** 12.3 "Невласні інтеграли" та 12.4 "Застосування ВІ...".

### Обґрунтування користі для IT-проєкту

Ця лекція розкриває величезний потенціал визначеного інтеграла для отримання корисних метрик, що виходять за рамки простого переміщення. Це дозволяє оцінити не тільки *чи* виконана вправа, але і *як* вона виконана.
-   **Обчислення шляху:** Інтеграл від **модуля швидкості (швидкості руху)** `s(t) = |v(t)|` дає **довжину шляху**, пройденого суглобом. Це чудовий показник ефективності руху. Якщо користувач досяг цілі, але при цьому багато рухався вперед-назад, довжина шляху буде значно більшою за переміщення.
-   **Обчислення "роботи" або "енергії":** Інтеграл від функції потужності `P(t)` (яка може залежати від сили, а отже, від маси та прискорення) дає повну **роботу**, виконану користувачем. Це можна перевести в спалені калорії, що є чудовим елементом гейміфікації.
-   **Невласні інтеграли:** Дозволяють моделювати процеси, що прямують до стабілізації "на нескінченності". Наприклад, можна оцінити сумарну помилку, якщо користувач намагатиметься утримувати позу нескінченно довго. Збіжність чи розбіжність такого інтеграла покаже, чи є поза принципово стабільною для даного користувача.

### Зв'язок математичної формалізації із описом IT-проєкту

| Елемент опису IT-проєкту | Математична формалізація (з розділів 12.3, 12.4) |
| :--- | :--- |
| Повне переміщення | `Δp = ∫[t₁,t₂] v(t) dt`. |
| Повний пройдений шлях | `L = ∫[t₁,t₂] |v(t)| dt`. |
| Енергія, витрачена на вправу | `Work = ∫[t₁,t₂] P(t) dt`, де `P(t)` - потужність. |
| Сумарна помилка за весь час утримання пози | **Невласний інтеграл**: `TotalError = ∫[0,∞] error(t) dt`. |
| Площа фігури, описаної кінцівкою | **Обчислення площі** за допомогою визначеного інтеграла (формула 12.4.7). |

### Приклади математичного опису IT-проєкту

**1. Оцінка ефективності руху (Шлях vs Переміщення):**

-   **Вправа:** "Проведіть руку з точки А в точку Б".
-   Користувач веде руку не по прямій, а з коливаннями.
-   Програма розраховує дві величини:
    1.  **Переміщення:** `Δp = P(Б) - P(А)`. Це просто різниця координат.
    2.  **Шлях:** `L = ∫[t_A,t_B] |v(t)| dt`. Для цього програма чисельно інтегрує модуль швидкості за весь час руху.
-   **Аналіз:** Нехай переміщення `Δp = 50 см`, а шлях `L = 75 см`.
-   **Рекомендація:** "Ви досягли цілі, але ваш рух був неоптимальним. Спробуйте вести руку більш прямо, щоб шлях був ближчим до 50 см".

**2. Підрахунок "балів енергії" (Площа під кривою):**

-   **Гейміфікація:** Програма нараховує бали за виконану роботу.
-   Програма моделює потужність `P(t)`, яку розвиває користувач, наприклад, як функцію, пропорційну квадрату швидкості: `P(t) = k * v(t)²`.
-   **Аналіз:** За час вправи `[t₁, t₂]` програма обчислює загальну енергію як визначений інтеграл `Energy = ∫[t₁,t₂] k*v(t)² dt`.
-   **Результат:** "Ви чудово попрацювали і витратили `N` одиниць енергії (спалили `X` калорій)!". Це пряма аналогія з геометричним змістом інтеграла як площі під графіком функції `P(t)`.

**3. Аналіз стабільності пози (Невласні інтеграли):**

-   **Завдання:** Утримувати складну балансову позу.
-   Програма відстежує функцію відхилення від ідеального балансу, `error(t)`. Зазвичай, з часом помилка зменшується, наприклад, за законом `error(t) = A*e⁻ᵏᵗ` або `A/(1+t²)`.
-   **Аналіз:** Програма може оцінити "сумарну нестабільність" за весь час, обчисливши невласний інтеграл `TotalError = ∫[0,∞] error(t) dt`.
-   Якщо цей інтеграл **збігається** (має скінченне значення), це означає, що користувач успішно стабілізував позу, і сума його помилок є скінченною.
-   Якщо інтеграл **розбігається**, це означає, що помилка згасає занадто повільно, і поза є фундаментально нестабільною для користувача на даному етапі.
-   **Рекомендація:** "Ви добре стабілізувалися, сумарна помилка мінімальна" або "Вам важко утримувати цю позу. Спробуйте простіший варіант для початку".
