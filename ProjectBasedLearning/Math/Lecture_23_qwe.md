## Обгрунтування користі від використання змісту лекції та практичного заняття для успішного виконання IT-проєкту:

У системі контролю рухів людини під час виконання йога-вправ, застосування визначеного інтеграла має важливе значення для аналізу та обробки даних про рух. Визначений інтеграл дозволяє обчислювати загальні характеристики руху, такі як загальний шлях, площі, об'єми та енергії, що важливо для кількісної оцінки якості виконання вправ.

Обчислення площі під кривою дозволяє аналізувати загальну активність руху протягом певного часу. Обчислення довжини дуги траєкторії руху суглоба дозволяє визначати загальну відстань, яку подолав суглоб під час виконання вправи. Це важливо для оцінки повноти виконання руху.

Практичне заняття, присвячене застосуванню визначеного інтеграла до задач геометрії та фізики, має безпосереднє застосування до аналізу рухів тіла. На практиці студенти навчаються обчислювати площі, довжини кривих та інші параметри, що моделюють рухи тіла, що необхідно для аналізу даних системи відстеження.

## Зв'язок математичної формалізації із описом IT-проєкту:

У системі розпізнавання рухів, при аналізі траєкторій руху суглобів часто виникають задачі, що потребують застосування визначеного інтеграла:

1. Обчислення площі під кривою швидкості для визначення загального переміщення
2. Обчислення довжини траєкторії руху суглоба
3. Обчислення об'єму тіла за площею поперечного перерізу для аналізу пози тіла
4. Обчислення енергії руху за відомою функцією сили або прискорення

Практичне заняття навчає студентів застосовувати визначений інтеграл до реальних задач, що моделюють рухи тіла, що необхідно для обробки даних системи відстеження.

Наприклад, при обчисленні загального шляху, пройденого суглобом:
s = ∫ |v(t)| dt

де v(t) - векторна функція швидкості суглоба.

## Приклади математичного опису IT-проєкту:

### Приклад 1: Обчислення загального шляху руху суглоба
Нехай функція швидкості суглоба задана як:
v(t) = (vx(t), vy(t), vz(t)) = (2t, cos(t), sin(t))

Для обчислення загального шляху, пройденого суглобом за інтервал часу [0, π]:
s = ∫₀^π ||v(t)|| dt = ∫₀^π √[(2t)² + cos²(t) + sin²(t)] dt = ∫₀^π √[4t² + 1] dt

Цей інтеграл обчислюється за допомогою підстановки:
u = 2t, du = 2dt, t = u/2
∫ √[u² + 1] (du/2) = (1/2)∫ √[u² + 1] du

Використовуючи формулу: ∫ √[x² + a²] dx = (x/2)√[x² + a²] + (a²/2)ln|x + √[x² + a²]| + C
∫₀^π √[4t² + 1] dt = (1/2)[(2t/2)√[4t² + 1] + (1/2)ln|2t + √[4t² + 1]|]₀^π
= (1/2)[t√[4t² + 1] + (1/2)ln|2t + √[4t² + 1]|]₀^π

Це дозволяє обчислити загальну довжину траєкторії руху суглоба за певний час.

### Приклад 2: Обчислення площі під кривою активності
При аналізі активності руху, якщо функція a(t) = 3t² + 2t + 1 описує рівень активності руху в певний момент часу, то загальна активність за інтервал [0, T]:
A = ∫₀^T (3t² + 2t + 1) dt = [t³ + t² + t]₀^T = T³ + T² + T

Це дозволяє кількісно оцінити загальну активність користувача під час виконання вправи.

### Приклад 3: Обчислення довжини кривої траєкторії (практичне заняття)
Для параметрично заданої траєкторії суглоба:
x(t) = cos(t), y(t) = sin(t), z(t) = t, де t ∈ [0, 2π]

Довжина кривої обчислюється за формулою:
L = ∫₀^(2π) √[(dx/dt)² + (dy/dt)² + (dz/dt)²] dt
= ∫₀^(2π) √[(-sin(t))² + (cos(t))² + 1²] dt
= ∫₀^(2π) √[sin²(t) + cos²(t) + 1] dt
= ∫₀^(2π) √2 dt = √2 × 2π = 2π√2

Це дозволяє обчислити загальну довжину траєкторії руху суглоба, що важливо для аналізу ефективності виконання вправи.

### Приклад 4: Обчислення об'єму тіла за площею поперечного перерізу (практичне заняття)
При аналізі пози тіла, якщо площа поперечного перерізу тіла визначається функцією A(h) = π(1 - h²), де h - висота перерізу, то об'єм тіла:
V = ∫₋₁^¹ A(h) dh = ∫₋₁^¹ π(1 - h²) dh = π[h - h³/3]₋₁^¹ = π[(1 - 1/3) - (-1 + 1/3)] = π[2/3 + 2/3] = 4π/3

Це дозволяє аналізувати об'ємні характеристики пози тіла під час виконання вправи.

### Приклад 5: Обчислення енергії руху (практичне заняття)
Якщо сила, що діє на суглоб, змінюється за законом F(t) = 2sin(t) + cos(2t), то робота (енергія) за інтервал [0, π]:
W = ∫₀^π F(t) dt = ∫₀^π (2sin(t) + cos(2t)) dt = [-2cos(t) + (1/2)sin(2t)]₀^π
= [-2cos(π) + (1/2)sin(2π)] - [-2cos(0) + (1/2)sin(0)]
= [-2(-1) + 0] - [-2(1) + 0] = [2] - [-2] = 4

Це дозволяє обчислити енергетичні характеристики руху, що важливо для аналізу ефективності виконання вправи.

### Приклад 6: Невласні інтеграли для аналізу тривалих рухів
Для аналізу тривалих рухів, якщо функція активності має вигляд f(t) = e^(-kt), де k > 0, то загальна активність за нескінченний час:
∫₀^∞ e^(-kt) dt = lim(T→∞) ∫₀^T e^(-kt) dt = lim(T→∞) [-1/k × e^(-kt)]₀^T = lim(T→∞) [-1/k × e^(-kT) + 1/k] = 1/k

Це дозволяє аналізувати загальну активність за тривалий період, що важливо для оцінки тривалості та інтенсивності виконання вправ.

Ці математичні поняття дозволяють системі аналізувати кількісні характеристики рухів, обчислювати енергетичні параметри та визначати оптимальні траєкторії для виконання йога-вправ.
