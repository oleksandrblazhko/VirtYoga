## Назва лекції: Числові, функціональні та степеневі ряди

## Назва практичного заняття: Збіжність рядів. Розкладання функцій в степеневі ряди. Деякі застосування степеневих рядів

## Обгрунтування користі від використання змісту лекції та практичного заняття для успішного виконання IT-проєкту:

У системі контролю рухів людини під час виконання йога-вправ, поняття числових, функціональних та степеневих рядів мають важливе значення для аналізу та обробки даних про рух. Степеневі ряди особливо корисні для апроксимації складних функцій, що описують рухи тіла, і для наближених обчислень значень функцій, які виникають при аналізі пози та руху користувача.

Практичне заняття, присвячене збіжності рядів, розкладанню функцій в степеневі ряди та їх застосуванню, має безпосереднє застосування до аналізу даних, що надходять від системи відстеження рухів. На практиці студенти навчаються розкладати функції в ряди Тейлора-Маклорена, що необхідно для аналізу складних функцій, що моделюють рухи тіла, а також для наближених обчислень, що важливі для швидкої обробки даних у реальному часі.

Збіжність рядів важлива для забезпечення стабільності обчислень у системі, оскільки розбіжність може призвести до некоректної обробки даних. Застосування степеневих рядів дозволяє системі ефективно обчислювати значення тригонометричних, експоненційних та інших функцій, що використовуються при аналізі рухів.

## Зв'язок математичної формалізації із описом IT-проєкту:

У системі розпізнавання рухів, при аналізі траєкторій руху суглобів часто виникають функції, що потребують застосування рядів для їх аналізу:

1. Тригонометричні функції, що описують кути між суглобами, можуть бути розкладені в ряди Тейлора:
   - sin(x) = x - x³/3! + x⁵/5! - ...
   - cos(x) = 1 - x²/2! + x⁴/4! - ...

2. Експоненційні функції, що моделюють згасання або зростання певних параметрів:
   - eˣ = 1 + x + x²/2! + x³/3! + ...

3. Степеневі функції, що описують залежності між параметрами руху:
   - (1+x)ⁿ = 1 + nx + n(n-1)x²/2! + ...

Практичне заняття навчає студентів застосовувати ці розклади для наближених обчислень, що важливо для швидкої обробки даних у реальному часі.

Для аналізу збіжності рядів, що моделюють параметри руху, використовуються ознаки збіжності:
- Ознака порівняння
- Ознака Даламбера
- Ознака Коші
- Ознака Лейбніца для знакозмінних рядів

Ці ознаки дозволяють визначити, чи обчислення параметрів руху буде стабільним і не призведе до розбіжності.

## Приклади математичного опису IT-проєкту:

### Приклад 1: Розкладання тригонометричних функцій у ряди для обчислення кутів
При аналізі кута між суглобами часто використовуються функції arcsin, arccos, arctan. Для обчислення цих функцій у реальному часі можна використовувати їх розклад у степеневі ряди:

arctan(x) = x - x³/3 + x⁵/5 - x⁷/7 + ... (для |x| < 1)

Це дозволяє системі швидко обчислювати кути між векторами, що з'єднують суглоби, без використання складних тригонометричних функцій.

### Приклад 2: Застосування ознаки Даламбера для аналізу збіжності ряду
Для аналізу стабільності параметрів руху, розглянемо ряд:
Σ(n=1,∞) xⁿ/n!

Для перевірки збіжності застосуємо ознаку Даламбера:
lim(n→∞) |aₙ₊₁/aₙ| = lim(n→∞) |xⁿ⁺¹/(n+1)! × n!/xⁿ| = lim(n→∞) |x/(n+1)| = 0 < 1

Оскільки границя менше 1, ряд збігається для будь-якого x. Це відповідає функції eˣ, що важливо для моделювання експоненційних залежностей у русі тіла.

### Приклад 3: Наближене обчислення значень функцій за допомогою рядів (практичне заняття)
Для швидкого обчислення значення функції sin(x) при малих кутах, можна використовувати початкові члени ряду:
sin(x) ≈ x - x³/6

Це дозволяє системі швидко обчислювати тригонометричні функції при аналізі малих відхилень від еталонної пози, що важливо для обробки даних у реальному часі.

### Приклад 4: Ряд для обчислення відстані між суглобами
При обчисленні відстані між двома суглобами, якщо відомі координати (x₁, y₁, z₁) і (x₂, y₂, z₂), виникає необхідність обчислення:
d = √[(x₂-x₁)² + (y₂-y₁)² + (z₂-z₁)²]

Якщо відстань мала, можна використовувати біноміальний ряд:
√(1+u) ≈ 1 + u/2 - u²/8 + ... (для |u| << 1)

Це дозволяє системі швидко обчислювати мінімальні зміни відстані між суглобами, що важливо для аналізу малих корекцій пози.

### Приклад 5: Застосування ознаки порівняння для аналізу збіжності
При аналізі ряду, що моделює накопичення помилок у вимірах:
Σ(n=1,∞) 1/n²

Цей ряд можна порівняти з відомим збіжним рядом (узагальнений гармонійний ряд з p=2 > 1), що гарантує стабільність обчислень при накопиченні помилок.

### Приклад 6: Розкладання функції в степеневий ряд для апроксимації (практичне заняття)
Для функції, що описує зміну кута між суглобами:
f(x) = ln(1+x) = x - x²/2 + x³/3 - x⁴/4 + ...

Цей ряд збігається при |x| < 1 і може бути використаний для аналізу логарифмічних змін параметрів руху, що важливо для аналізу енергетичних характеристик рухів.

Ці математичні поняття дозволяють системі ефективно обробляти дані про рухи, забезпечуючи стабільність обчислень і швидкість аналізу, що критично важливо для системи відстеження рухів у реальному часі.
