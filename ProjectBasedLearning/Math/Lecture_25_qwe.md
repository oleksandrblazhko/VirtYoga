## Обгрунтування користі від використання змісту лекції та практичного заняття для успішного виконання IT-проєкту:

У системі контролю рухів людини під час виконання йога-вправ, перетворення Фур'є та ряди Фур'є мають критично важливе значення для аналізу та обробки сигналів, що надходять від системи відстеження рухів. Ці математичні інструменти дозволяють розкласти складні рухові сигнали на прості гармонійні компоненти, що дозволяє аналізувати періодичність рухів, виявляти аномалії, фільтрувати шум та визначати основні частоти рухових патернів.

Перетворення Фур'є особливо корисне для аналізу даних, що надходять від веб-камери, оскільки воно дозволяє перейти від часових даних до частотної області, що важливо для виявлення регулярних рухів та відстеження їх стабільності. Це дозволяє системі визначати, чи користувач виконує вправи з певною ритмічністю та стабільністю, що є важливим аспектом йоги.

Практичне заняття, присвячене розкладанню парних і непарних функцій в ряди Фур'є, має безпосереднє застосування до аналізу симетричних рухів тіла. На практиці студенти навчаються розкладати функції, що моделюють рухи тіла, в тригонометричні ряди, що необхідно для аналізу симетрії пози та рухів, що важливо для визначення правильності виконання йога-вправ.

Спектральна функція дозволяє визначити, які частоти домінують у русі користувача, що корисно для аналізу плавності та стабільності виконання вправи.

## Зв'язок математичної формалізації із описом IT-проєкту:

У системі розпізнавання рухів, сигнали, що надходять від веб-камери, можуть бути представлені як функції часу, що описують зміну координат суглобів. Ці функції можна розкласти в ряд Фур'є:

f(t) = a₀/2 + Σ(n=1,∞) [aₙcos(nω₀t) + bₙsin(nω₀t)]

де ω₀ = 2π/T - основна кутова частота, T - період функції.

Для аналізу рухів користувача, кожен суглоб може бути представлений як сума гармонійних компонентів:
x(t) = Σ(n=0,∞) [Aₙcos(nω₀t + φₙ)]

де Aₙ - амплітуда n-го гармонійного компонента, φₙ - фаза.

Практичне заняття навчає студентів розкладати парні та непарні функції в ряди Фур'є, що застосовується до аналізу симетричних рухів тіла. Наприклад, якщо рух лівої та правої сторін тіла є симетричним, це може бути представлено як парна або непарна функція.

Для парної функції f(-t) = f(t), ряд Фур'є містить лише косинуси:
f(t) = a₀/2 + Σ(n=1,∞) aₙcos(nω₀t)

Для непарної функції f(-t) = -f(t), ряд Фур'є містить лише синуси:
f(t) = Σ(n=1,∞) bₙsin(nω₀t)

Це дозволяє ефективно аналізувати симетрію пози та рухів користувача.

## Приклади математичного опису IT-проєкту:

### Приклад 1: Розкладання руху суглоба в ряд Фур'є
Нехай x(t) - координата плечового суглоба як функція часу під час виконання певної йога-вправи. Для аналізу періодичності руху, можна розкласти цю функцію в ряд Фур'є:

x(t) = a₀/2 + Σ(n=1,∞) [aₙcos(nω₀t) + bₙsin(nω₀t)]

де коефіцієнти обчислюються за формулами:
a₀ = (2/T)∫₀ᵀ x(t)dt
aₙ = (2/T)∫₀ᵀ x(t)cos(nω₀t)dt
bₙ = (2/T)∫₀ᵀ x(t)sin(nω₀t)dt

Це дозволяє визначити основні частоти руху та виявити домінуючі гармонійні компоненти, що важливо для аналізу плавності та стабільності виконання вправи.

### Приклад 2: Застосування перетворення Фур'є для аналізу сигналу
Для аналізу неперіодичних рухів, використовується перетворення Фур'є:
F(ω) = ∫₋∞⁺∞ f(t)e⁻ⁱʷᵗdt

де f(t) - функція, що описує рух суглоба, F(ω) - її представлення в частотній області.

Це дозволяє аналізувати спектральні характеристики руху, виявляти шум та аномалії, що важливо для обробки даних системи відстеження рухів.

### Приклад 3: Аналіз симетрії пози (практичне заняття)
Якщо функція, що описує рух лівої руки f_left(t) є непарною відносно функції руху правої руки f_right(t), тобто f_left(t) = -f_right(t), то можна використовувати розклад в ряд Фур'є лише з синусами:

f(t) = Σ(n=1,∞) bₙsin(nω₀t)

де bₙ = (2/T)∫₀ᵀ f(t)sin(nω₀t)dt

Це дозволяє ефективно аналізувати симетрію пози та виявляти асиметрії, що важливо для правильного виконання йога-вправ.

### Приклад 4: Використання інтеграла Фур'є для аналізу неперіодичних рухів
Для аналізу неперіодичних рухів, що не мають чітко визначеного періоду, використовується інтеграл Фур'є:
f(t) = (1/2π)∫₋∞⁺∞ F(ω)eⁱʷᵗdω

де F(ω) - спектральна функція, що описує частотний вміст сигналу.

Це дозволяє аналізувати будь-які рухові сигнали, навіть ті, що не є строго періодичними, що важливо для аналізу різноманітних йога-вправ.

### Приклад 5: Спектральний аналіз для виявлення аномалій
Спектральна функція |F(ω)|² показує розподіл енергії сигналу за частотами. Якщо в певному діапазоні частот спостерігається різке зростання енергії, це може вказувати на неправильний рух або шум у даних.

Наприклад, для функції відхилення від еталонної позиції E(t):
E(ω) = ∫₋∞⁺∞ E(t)e⁻ⁱʷᵗdt

|E(ω)|² показує, на яких частотах відбуваються найбільші відхилення, що дозволяє визначити проблемні ділянки виконання вправи.

### Приклад 6: Застосування рядів Фур'є для фільтрації сигналу
Для фільтрації шуму з сигналів відстеження рухів, можна використовувати часткову суму ряду Фур'є:
Sₙ(t) = a₀/2 + Σ(k=1,N) [aₖcos(kω₀t) + bₖsin(kω₀t)]

де N - обраний порядок фільтра. Відкидання високочастотних компонентів (великі k) дозволяє згладити шум, зберігаючи основну інформацію про рух.

Це особливо корисно для обробки даних від веб-камери, де може бути присутній шум через обмежену роздільну здатність або освітлення.

### Приклад 7: Аналіз парних і непарних функцій для оцінки симетрії (практичне заняття)
Для аналізу симетрії пози тіла, можна розглядати різницю між лівою та правою стороною тіла як функцію:
g(t) = f_left(t) - f_right(t)

Якщо g(t) є непарною функцією (g(-t) = -g(t)), то вона може бути розкладена в ряд Фур'є тільки з синусами:
g(t) = Σ(n=1,∞) bₙsin(nω₀t)

де bₙ = (2/T)∫₀ᵀ g(t)sin(nω₀t)dt

Якщо коефіцієнти bₙ мають великі значення, це вказує на асиметрію в позі, що може свідчити про неправильне виконання вправи.

Ці математичні поняття дозволяють системі аналізувати рухи користувача в частотній області, виявляти аномалії, оцінювати симетрію та стабільність рухів, що критично важливо для системи контролю якості виконання йога-вправ.
