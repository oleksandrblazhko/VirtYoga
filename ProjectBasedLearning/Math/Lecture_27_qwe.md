## Обгрунтування користі від використання змісту лекції та практичного заняття для успішного виконання IT-проєкту:

У системі контролю рухів людини під час виконання йога-вправ, аналіз екстремумів функцій багатьох змінних має важливе значення для визначення оптимальних позицій та параметрів руху. Система повинна вміти визначати, коли певні параметри досягають максимальних або мінімальних значень, що важливо для аналізу якості виконання вправи.

Екстремуми функцій багатьох змінних дозволяють:
- Визначати оптимальні позиції суглобів для досягнення максимальної ефективності вправи
- Аналізувати мінімальні відхилення від еталонної пози, що вказує на правильне виконання
- Виявляти критичні точки, де відбувається зміна характеру руху
- Оптимізувати параметри руху для досягнення найкращого результату

Практичне заняття, присвячене знаходженню критичних точок і визначенню екстремуму функції багатьох змінних, має безпосереднє застосування до аналізу функцій, що моделюють рухи тіла. На практиці студенти навчаються обчислювати частинні похідні, знаходити стаціонарні точки та визначати тип екстремуму, що необхідно для аналізу даних системи відстеження.

## Зв'язок математичної формалізації із описом IT-проєкту:

У системі розпізнавання рухів, функції багатьох змінних використовуються для моделювання:
1. Функції помилки, що вимірює відхилення від еталонної пози: E = f(x₁, y₁, z₁, ..., xₙ, yₙ, zₙ)
2. Функції енергії руху: E = g(vx₁, vy₁, vz₁, ..., vxₙ, vyₙ, vzₙ)
3. Функції відстані між суглобами: d = h(x₁, y₁, z₁, x₂, y₂, z₂)

Для функції помилки E = f(x₁, y₁, z₁, ..., xₙ, yₙ, zₙ), мінімум функції відповідає найкращому збігу з еталонною позою. Знаходження критичних точок:
∂f/∂xᵢ = 0, ∂f/∂yᵢ = 0, ∂f/∂zᵢ = 0 для всіх i = 1, ..., n

Практичне заняття навчає студентів знаходити критичні точки і визначати тип екстремуму для таких функцій, що необхідно для аналізу якості виконання вправ.

## Приклади математичного опису IT-проєкту:

### Приклад 1: Функція помилки як функція багатьох змінних
Нехай є функція помилки для пози трьох суглобів:
E(x₁, y₁, z₁, x₂, y₂, z₂, x₃, y₃, z₃) = (x₁ - x₁*)² + (y₁ - y₁*)² + (z₁ - z₁*)² + (x₂ - x₂*)² + (y₂ - y₂*)² + (z₂ - z₂*)² + (x₃ - x₃*)² + (y₃ - y₃*)² + (z₃ - z₃*)²

де (xᵢ*, yᵢ*, zᵢ*) - еталонні координати i-го суглоба.

Для знаходження мінімуму:
∂E/∂x₁ = 2(x₁ - x₁*) = 0 ⇒ x₁ = x₁*
∂E/∂y₁ = 2(y₁ - y₁*) = 0 ⇒ y₁ = y₁*
∂E/∂z₁ = 2(z₁ - z₁*) = 0 ⇒ z₁ = z₁*
і т.д. для всіх координат.

Це означає, що мінімум функції помилки досягається, коли координати суглобів збігаються з еталонними.

### Приклад 2: Визначення типу екстремуму
Для функції E(x₁, y₁, z₁, x₂, y₂, z₂), другі частинні похідні:
∂²E/∂x₁² = 2, ∂²E/∂y₁² = 2, ∂²E/∂z₁² = 2, ∂²E/∂x₂² = 2, ∂²E/∂y₂² = 2, ∂²E/∂z₂² = 2

Матриця Гессе:
H = [2  0  0  0  0  0]
    [0  2  0  0  0  0]
    [0  0  2  0  0  0]
    [0  0  0  2  0  0]
    [0  0  0  0  2  0]
    [0  0  0  0  0  2]

Всі власні значення додатні (λ = 2), отже, знайдена точка є мінімумом.

### Приклад 3: Знаходження критичних точок (практичне заняття)
Для функції E(x, y) = x² + y² - 2xy - 2x - 2y:
∂E/∂x = 2x - 2y - 2 = 0
∂E/∂y = 2y - 2x - 2 = 0

Розв'язуючи систему:
2x - 2y = 2
-2x + 2y = 2

Додаємо рівняння: 0 = 4 - це суперечність, отже, переглядаємо:
2x - 2y - 2 = 0 ⇒ x - y = 1
-2x + 2y - 2 = 0 ⇒ -x + y = 1

З другого: y = x + 1, підставляємо в перше: x - (x + 1) = 1 ⇒ -1 = 1 - суперечність.

Перевіряємо: 2x - 2y - 2 = 0 і -2x + 2y - 2 = 0
2x - 2y = 2 і -2x + 2y = 2
Додаємо: 0 = 4 - немає розв'язку.

Отже, функція не має стаціонарних точок, що вказує на відсутність локального екстремуму.

### Приклад 4: Аналіз енергії руху
Функція енергії руху для двох суглобів:
E(x₁, y₁, z₁, x₂, y₂, z₂) = (vx₁)² + (vy₁)² + (vz₁)² + (vx₂)² + (vy₂)² + (vz₂)²

де vxᵢ, vyᵢ, vzᵢ - швидкості зміни координат.

Для мінімізації енергії руху (для плавності):
∂E/∂vx₁ = 2vx₁ = 0 ⇒ vx₁ = 0
∂E/∂vy₁ = 2vy₁ = 0 ⇒ vy₁ = 0
і т.д.

Це означає, що для мінімальної енергії рух має бути мінімально швидким.

### Приклад 5: Оптимізація кута між суглобами
Функція, що описує кут між векторами:
θ(x₁, y₁, z₁, x₂, y₂, z₂, x₃, y₃, z₃) = arccos((u·v)/(|u||v|))

де u = (x₂-x₁, y₂-y₁, z₂-z₁), v = (x₃-x₂, y₃-y₂, z₃-z₂)

Для знаходження екстремумів кута:
∂θ/∂x₁ = 0, ∂θ/∂y₁ = 0, ..., ∂θ/∂z₃ = 0

Це дозволяє визначити, при яких координатах кут досягає екстремальних значень, що важливо для аналізу правильності пози.

### Приклад 6: Визначення стаціонарних точок (практичне заняття)
Для функції f(x, y) = x³ + y³ - 3xy:
∂f/∂x = 3x² - 3y = 0
∂f/∂y = 3y² - 3x = 0

З першого: y = x², підставляємо в друге: 3(x²)² - 3x = 0 ⇒ 3x⁴ - 3x = 0 ⇒ 3x(x³ - 1) = 0

Отже, x = 0 або x = 1
Якщо x = 0, то y = 0
Якщо x = 1, то y = 1

Критичні точки: (0,0) і (1,1)

Для визначення типу екстремуму:
∂²f/∂x² = 6x, ∂²f/∂y² = 6y, ∂²f/∂x∂y = -3

У точці (0,0): A = 0, B = -3, C = 0
D = AC - B² = 0 - 9 = -9 < 0, отже, (0,0) - сідлова точка

У точці (1,1): A = 6, B = -3, C = 6
D = 36 - 9 = 27 > 0, A = 6 > 0, отже, (1,1) - мінімум

Ці математичні поняття дозволяють системі аналізувати екстремальні значення параметрів руху, визначати оптимальні позиції та виявляти критичні точки для забезпечення якості виконання йога-вправ.
