## Обгрунтування користі від використання змісту лекції та практичного заняття для успішного виконання IT-проєкту:

У системі контролю рухів людини під час виконання йога-вправ, умовні екстремуми мають важливе значення для оптимізації рухів з урахуванням обмежень. Наприклад, при аналізі пози тіла необхідно враховувати анатомічні обмеження, які не дозволяють суглобам приймати будь-які положення. Ці обмеження можна формалізувати як рівняння зв'язку, і тоді задача оптимізації пози зводиться до задачі на умовний екстремум.

Метод Лагранжа дозволяє знаходити оптимальні параметри руху при наявності обмежень, що важливо для забезпечення безпечного виконання вправ. Метод підстановки також корисний для аналізу ситуацій, коли один параметр залежить від іншого.

Практичне заняття, присвячене методу найменших квадратів, має безпосереднє застосування до аналізу експериментальних даних, що надходять від системи відстеження рухів. На практиці студенти навчаються будувати емпіричні формули, що описують залежності між параметрами руху, що необхідно для аналізу даних системи відстеження.

## Зв'язок математичної формалізації із описом IT-проєкту:

У системі розпізнавання рухів, задачі оптимізації часто мають обмеження, що відображають фізичні або анатомічні обмеження тіла. Наприклад:

1. Обмеження на кути між суглобами: φ₁ ≤ φ ≤ φ₂
2. Обмеження на відстані між суглобами: ||Sᵢ - Sⱼ|| ≤ dₘₐₓ
3. Обмеження на площу опори: (x, y) ∈ D, де D - область, в якій людина зберігає рівновагу

Для задачі мінімізації функції помилки E(x₁, y₁, z₁, ..., xₙ, yₙ, zₙ) при наявності обмежень gᵢ(x₁, y₁, z₁, ..., xₙ, yₙ, zₙ) = 0, використовується метод Лагранжа:
L = E + λ₁g₁ + λ₂g₂ + ... + λₘgₘ

Практичне заняття навчає студентів застосовувати метод найменших квадратів для апроксимації експериментальних даних, що важливо для аналізу та моделювання рухів тіла.

## Приклади математичного опису IT-проєкту:

### Приклад 1: Умовний екстремум для оптимізації пози
Знайти мінімум функції помилки E(x, y, z) = (x - x*)² + (y - y*)² + (z - z*)² при обмеженні, що точка (x, y, z) лежить на сфері радіуса R з центром у початку координат: x² + y² + z² = R².

Функція Лагранжа:
F(x, y, z, λ) = (x - x*)² + (y - y*)² + (z - z*)² + λ(x² + y² + z² - R²)

Умови стаціонарності:
∂F/∂x = 2(x - x*) + 2λx = 0
∂F/∂y = 2(y - y*) + 2λy = 0
∂F/∂z = 2(z - z*) + 2λz = 0
∂F/∂λ = x² + y² + z² - R² = 0

Це дозволяє знайти найближчу точку на сфері до еталонної точки (x*, y*, z*), що важливо для аналізу пози тіла.

### Приклад 2: Метод підстановки для аналізу руху
Якщо відомо, що два суглоби повинні залишатися на сталій відстані d, тобто (x₁-x₂)² + (y₁-y₂)² + (z₁-z₂)² = d², то можна використовувати метод підстановки.

Наприклад, якщо z₂ = z₁ + h, де h - стала висота, то:
(x₁-x₂)² + (y₁-y₂)² + h² = d²
(x₁-x₂)² + (y₁-y₂)² = d² - h²

Це дозволяє звести задачу з трьох змінних до двох, що спрощує аналіз.

### Приклад 3: Знаходження найбільшого та найменшого значень у обмеженій області
Для аналізу руху в певній області D, необхідно знайти екстремуми функції на межі області. Наприклад, для функції E(x, y) = x² + y² - xy - x - y в трикутнику D = {(x, y) | x ≥ 0, y ≥ 0, x + y ≤ 1}:

1. Знаходимо внутрішні критичні точки: ∂E/∂x = 2x - y - 1 = 0, ∂E/∂y = 2y - x - 1 = 0
   Розв'язок: x = y = 1, але точка (1,1) не належить D

2. Досліджуємо поведінку на межі:
   - x = 0: E(0, y) = y² - y, 0 ≤ y ≤ 1
   - y = 0: E(x, 0) = x² - x, 0 ≤ x ≤ 1
   - x + y = 1: E(x, 1-x) = x² + (1-x)² - x(1-x) - x - (1-x)

Це дозволяє визначити межі, в яких параметри руху залишаються при виконанні вправи.

### Приклад 4: Метод найменших квадратів для апроксимації даних (практичне заняття)
При аналізі даних відстеження рухів, може знадобитися знайти лінійну залежність між координатами суглоба та часом: z = at + b.

Для набору точок (tᵢ, zᵢ), i = 1, ..., n, метод найменших квадратів мінімізує суму:
S = Σ(zᵢ - (atᵢ + b))²

Умови мінімуму:
∂S/∂a = -2Σ(zᵢ - (atᵢ + b))tᵢ = 0
∂S/∂b = -2Σ(zᵢ - (atᵢ + b)) = 0

Це дає систему рівнянь для знаходження параметрів a і b, що описують лінійну тенденцію руху суглоба.

### Приклад 5: Аналіз умовного екстремуму для кутів між суглобами
Для аналізу кута між трьома суглобами при обмеженні на довжину відрізків між ними:
Нехай є три суглоби A(x₁, y₁, z₁), B(x₂, y₂, z₂), C(x₃, y₃, z₃), і відомі відстані |AB| = d₁, |BC| = d₂.

Потрібно знайти екстремум кута ABC при цих обмеженнях:
f = (AB·BC)/(|AB||BC|) = ((x₂-x₁)(x₃-x₂) + (y₂-y₁)(y₃-y₂) + (z₂-z₁)(z₃-z₂))/(d₁d₂)

Обмеження:
(x₂-x₁)² + (y₂-y₁)² + (z₂-z₁)² = d₁²
(x₃-x₂)² + (y₃-y₂)² + (z₃-z₂)² = d₂²

Це дозволяє аналізувати оптимальні кути між суглобами при виконанні вправи.

### Приклад 6: Застосування методу Лагранжа до оптимізації енергії руху
Для мінімізації енергії руху при обмеженні на траєкторію:
E = ∫[vₓ²(t) + vᵧ²(t) + v_z²(t)]dt → min
при умові, що траєкторія проходить через певні контрольні точки.

Це дозволяє знаходити найбільш енергоефективні траєкторії руху суглобів при виконанні йога-вправ.

Ці математичні поняття дозволяють системі аналізувати умовні екстремуми параметрів руху, враховувати обмеження тіла та знаходити оптимальні траєкторії для виконання йога-вправ.
