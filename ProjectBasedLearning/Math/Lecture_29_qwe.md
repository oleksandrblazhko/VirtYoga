# Сценарій використання математичного опису для Лекції 29 та Практичного заняття 29: Звичайні диференціальні рівняння. Типи рівнянь першого порядку і методи їх розв'язку. Розв'язки диференціальних рівнянь першого порядку. Задача Коші для ЛНДР першого порядку

## Номер лекції: 29
## Назва лекції: Звичайні диференціальні рівняння. Типи рівнянь першого порядку і методи їх розв'язку

## Назва практичного заняття: Розв'язки диференціальних рівнянь першого порядку. Задача Коші для ЛНДР першого порядку

## Обгрунтування користі від використання змісту лекції та практичного заняття для успішного виконання IT-проєкту:

У системі контролю рухів людини під час виконання йога-вправ, звичайні диференціальні рівняння першого порядку мають важливе значення для моделювання динаміки рухів та аналізу зміни параметрів руху залежно від часу. Диференціальні рівняння дозволяють описати, як швидкість зміни положення суглобів залежить від самого положення, що критично важливо для аналізу плавності рухів та передбачення майбутніх позицій.

Рівняння з відокремлюваними змінними особливо корисні для моделювання ситуацій, коли швидкість зміни кута між суглобами залежить від самого кута. Однорідні диференціальні рівняння можуть застосовуватися до аналізу пропорційних змін між різними параметрами руху. Лінійні рівняння першого порядку важливі для моделювання процесів з вхідними параметрами, що змінюються з часом.

Практичне заняття, присвячене розв'язкам диференціальних рівнянь першого порядку та задачі Коші, має безпосереднє застосування до аналізу даних, що надходять від системи відстеження рухів. На практиці студенти навчаються розв'язувати рівняння, що моделюють рухи тіла, а також вирішувати задачі Коші, що необхідно для прогнозування рухів на основі початкових умов.

Задача Коші особливо важлива для системи, оскільки дозволяє передбачити, як буде розвиватися рух користувача на основі його поточного стану, що необхідно для надання рекомендацій щодо корекції пози.

## Зв'язок математичної формалізації із описом IT-проєкту:

У системі розпізнавання рухів, диференціальні рівняння першого порядку можуть бути застосовані до моделювання:

1. Швидкості зміни кута між суглобами: dθ/dt = f(θ, t)
2. Зміни відстані між суглобами: dr/dt = g(r, t)
3. Процесу наближення до еталонної пози: dx/dt = h(x, x_ref)

Для аналізу руху суглоба можна використовувати рівняння типу:
dx/dt = k(x_ref - x)

де x - поточна позиція суглоба, x_ref - еталонна позиція, k - коефіцієнт, що визначає швидкість корекції.

Практичне заняття навчає студентів розв'язувати такі рівняння, що необхідно для аналізу динаміки рухів та передбачення майбутніх позицій.

Для рівняння з відокремлюваними змінними:
dy/dx = g(x)/h(y)

розв'язок полягає у відокремленні змінних та інтегруванні:
∫ h(y)dy = ∫ g(x)dx + C

Це дозволяє аналізувати залежності між різними параметрами руху.

## Приклади математичного опису IT-проєкту:

### Приклад 1: Моделювання зміни кута між суглобами
Розглянемо модель, що описує зміну кута між суглобами при виконанні йога-вправи:
dθ/dt = -k(θ - θ_ref)

де θ - поточний кут між суглобами, θ_ref - еталонний кут, k - додатна стала, що визначає швидкість наближення до еталону.

Це лінійне диференціальне рівняння першого порядку з розв'язком:
θ(t) = θ_ref + (θ_0 - θ_ref)e^(-kt)

де θ_0 - початкове значення кута.

Це дозволяє системі прогнозувати, як швидко користувач наближається до правильної пози, і визначати, чи виконує він вправу з необхідною плавністю.

### Приклад 2: Рівняння з відокремлюваними змінними для аналізу відстані
Для аналізу зміни відстані між двома суглобами, можна використовувати рівняння:
dr/dt = -a·r + b

де r - відстань між суглобами, a, b - сталі параметри.

Відокремлюємо змінні:
dr/(a·r - b) = -dt

Інтегруємо:
(1/a)ln|a·r - b| = -t + C
ln|a·r - b| = -a·t + C₁
a·r - b = C₂e^(-a·t)
r(t) = (b + C₂e^(-a·t))/a

З урахуванням початкової умови r(0) = r₀:
r(t) = (b + (a·r₀ - b)e^(-a·t))/a

Це дозволяє моделювати, як змінюється відстань між суглобами залежно від часу, що важливо для аналізу правильності виконання певних поз.

### Приклад 3: Однорідне диференціальне рівняння для аналізу пропорційних змін
Для аналізу ситуацій, коли зміна одного параметра пропорційна зміні іншого, можна використовувати однорідне рівняння:
dy/dx = f(y/x)

Наприклад, якщо зміна координати y суглоба відносно зміни координати x має певну пропорційність:
dy/dx = (y² - x²)/(2xy)

Це однорідне рівняння, яке можна розв'язати заміною v = y/x, тобто y = vx, dy/dx = v + x·dv/dx:
v + x·dv/dx = (v²x² - x²)/(2vx²) = (v² - 1)/(2v)
x·dv/dx = (v² - 1)/(2v) - v = (v² - 1 - 2v²)/(2v) = -(v² + 1)/(2v)
(2v)/(v² + 1)dv = -dx/x

Інтегруємо:
ln(v² + 1) = -ln|x| + C
v² + 1 = C₁/|x|
(y/x)² + 1 = C₁/|x|
y² + x² = C₁·|x|

Це рівняння описує траєкторію руху суглоба, що важливо для аналізу певних йога-поз.

### Приклад 4: Задача Коші для моделювання руху (практичне заняття)
Розглянемо задачу Коші для моделювання зміни кута нахилу тіла:
dθ/dt = -k·sin(θ), θ(0) = θ₀

де k - параметр, що визначає швидкість зміни.

Це рівняння з відокремлюваними змінними:
dθ/sin(θ) = -k·dt

Інтегруємо ліву частину:
∫ csc(θ)dθ = -k·t + C
ln|tan(θ/2)| = -k·t + C

З початкової умови θ(0) = θ₀:
ln|tan(θ₀/2)| = C

Отже:
ln|tan(θ/2)| = -k·t + ln|tan(θ₀/2)|
tan(θ/2) = tan(θ₀/2)·e^(-k·t)
θ(t) = 2·arctan(tan(θ₀/2)·e^(-k·t))

Це дозволяє системі передбачити, як буде змінюватися кут нахилу тіла залежно від часу, що важливо для аналізу стабільності пози.

### Приклад 5: Лінійне диференціальне рівняння для аналізу відхилення
Для аналізу відхилення пози від еталонної, можна використовувати лінійне рівняння:
dx/dt + p(t)x = q(t)

де x - відхилення від еталонної позиції, p(t) - функція, що визначає швидкість зменшення відхилення, q(t) - функція, що враховує зовнішні впливи.

Наприклад:
dx/dt + 2x = sin(t)

Це лінійне рівняння з інтегруючим множником μ(t) = e^(∫2dt) = e^(2t):
d/dt[x·e^(2t)] = sin(t)·e^(2t)
x·e^(2t) = ∫sin(t)·e^(2t)dt + C

Використовуючи інтегрування частинами двічі:
∫sin(t)·e^(2t)dt = (e^(2t)/5)(2sin(t) - cos(t)) + C

Отже:
x(t) = (2sin(t) - cos(t))/5 + C·e^(-2t)

З початкової умови x(0) = x₀:
x₀ = -1/5 + C
C = x₀ + 1/5

Таким чином:
x(t) = (2sin(t) - cos(t))/5 + (x₀ + 1/5)e^(-2t)

Це дозволяє моделювати, як змінюється відхилення користувача від еталонної пози залежно від часу.

### Приклад 6: Рівняння в повних диференціалах для аналізу енергії руху
Для аналізу енергії руху можна використовувати рівняння в повних диференціалах:
M(x,y)dx + N(x,y)dy = 0

де x - позиція суглоба, y - швидкість зміни позиції.

Наприклад:
(2xy + 3x²)dx + (x² + 2y)dy = 0

Перевіряємо умову повного диференціала:
∂M/∂y = 2x, ∂N/∂x = 2x

Оскільки ∂M/∂y = ∂N/∂x, рівняння є рівнянням в повних диференціалах.

Шукаємо функцію u(x,y) таку, що:
∂u/∂x = 2xy + 3x², ∂u/∂y = x² + 2y

Інтегруємо перше рівняння:
u = ∫(2xy + 3x²)dx = x²y + x³ + φ(y)

Диференціюємо по y:
∂u/∂y = x² + φ'(y) = x² + 2y

Отже, φ'(y) = 2y, φ(y) = y² + C

Загальний розв'язок:
u(x,y) = x²y + x³ + y² = C

Це рівняння зберігання енергії руху, що важливо для аналізу стабільності та безпечності виконання вправ.

Ці математичні поняття дозволяють системі моделювати динаміку рухів, прогнозувати майбутні позиції та аналізувати стабільність виконання йога-вправ, забезпечуючи ефективне керування процесом навчання.
