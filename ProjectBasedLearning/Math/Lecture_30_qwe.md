## Обгрунтування користі від використання змісту лекції та практичного заняття для успішного виконання IT-проєкту:

У системі контролю рухів людини під час виконання йога-вправ, диференціальні рівняння вищих порядків та системи диференціальних рівнянь мають критично важливе значення для моделювання динаміки рухів та аналізу прискорення. Друга похідна (прискорення) є ключовим параметром для аналізу плавності рухів та виявлення різких змін, що можуть вказувати на неправильне виконання вправи або ризик травми.

Системи диференціальних рівнянь особливо корисні для моделювання взаємопов'язаних рухів різних суглобів, де зміна положення одного суглоба впливає на інші. Це дозволяє системі аналізувати комплексні рухові патерни, що важливо для оцінки якості виконання йога-вправ.

Практичне заняття, присвячене методам розв'язку диференціальних рівнянь вищих порядків і систем ЗДУ першого порядку, має безпосереднє застосування до аналізу даних, що надходять від системи відстеження рухів. На практиці студенти навчаються розв'язувати системи рівнянь, що моделюють взаємопов'язані рухи тіла, що необхідно для аналізу даних системи відстеження.

## Зв'язок математичної формалізації із описом IT-проєкту:

У системі розпізнавання рухів, рух кожного суглоба може бути описаний системою диференціальних рівнянь вищих порядків:

Для кожного суглоба з координатами (x, y, z):
- x''(t) = f₁(x, y, z, x', y', z', t) - рівняння для прискорення за x
- y''(t) = f₂(x, y, z, x', y', z', t) - рівняння для прискорення за y
- z''(t) = f₃(x, y, z, x', y', z', t) - рівняння для прискорення за z

Це утворює систему з 3 диференціальних рівнянь другого порядку, що може бути зведена до системи з 6 рівнянь першого порядку шляхом введення нових змінних:
- x₁ = x, x₂ = x', x₃ = y, x₄ = y', x₅ = z, x₆ = z'

Тоді система перетворюється на:
- x₁' = x₂
- x₂' = f₁(x₁, x₃, x₅, x₂, x₄, x₆, t)
- x₃' = x₄
- x₄' = f₂(x₁, x₃, x₅, x₂, x₄, x₆, t)
- x₅' = x₆
- x₆' = f₃(x₁, x₃, x₅, x₂, x₄, x₆, t)

Практичне заняття навчає студентів методам розв'язку таких систем, що необхідно для аналізу складних рухових патернів у системі відстеження.

## Приклади математичного опису IT-проєкту:

### Приклад 1: Моделювання руху суглоба за допомогою диференціального рівняння другого порядку
Розглянемо модель руху плечового суглоба, яка враховує демпфування та пружність:
mx''(t) + cx'(t) + kx(t) = F(t)

де:
- m - маса суглоба (у віртуальному сенсі)
- c - коефіцієнт демпфування
- k - коефіцієнт пружності (відхилення від еталонної пози)
- F(t) - зовнішня сила (рух користувача)
- x(t) - відхилення суглоба від еталонної позиції

Це диференціальне рівняння другого порядку описує, як суглоб реагує на зовнішній рух користувача, враховуючи інерційність та пружність тіла.

### Приклад 2: Система диференціальних рівнянь для моделювання взаємопов'язаних рухів
Для моделювання взаємопов'язаних рухів плеча та ліктя, можна використовувати систему:
x₁''(t) = -k₁x₁(t) - k₂(x₁(t) - x₂(t)) + u₁(t)
x₂''(t) = -k₃x₂(t) + k₂(x₁(t) - x₂(t)) + u₂(t)

де:
- x₁(t) - положення плечового суглоба
- x₂(t) - положення ліктьового суглоба
- k₁, k₂, k₃ - коефіцієнти жорсткості
- u₁(t), u₂(t) - керуючі впливи (рухи користувача)

Ця система дозволяє моделювати, як рух одного суглоба впливає на інший, що важливо для аналізу координації рухів.

### Приклад 3: Застосування методу Лагранжа для розв'язку неоднорідної системи (практичне заняття)
Для системи:
x'(t) = 2x(t) + y(t) + e^t
y'(t) = x(t) + 2y(t) + 2e^t

Практичне заняття навчає студентів розв'язувати такі системи методом Лагранжа. Спочатку розв'язується однорідна система, а потім знаходиться частинний розв'язок неоднорідної системи.

Це застосовується до моделювання рухів, де є як внутрішні взаємодії між суглобами, так і зовнішні впливи (наприклад, команди системи або зворотній зв'язок).

### Приклад 4: Аналіз стійкості рухової системи
Для аналізу стійкості пози тіла, можна використовувати систему рівнянь:
X'(t) = AX(t) + BU(t)

де X(t) - вектор стану (координати та швидкості суглобів), A - матриця системи, B - матриця впливу, U(t) - вектор керування.

Стійкість системи визначається власними значеннями матриці A. Якщо всі власні значення мають від'ємні дійсні частини, система стійка, що означає, що невеликі відхилення від еталонної пози будуть зменшуватися з часом.

### Приклад 5: Застосування методу зниження порядку для аналізу руху
Для диференціального рівняння третього порядку, що описує рух суглоба:
x'''(t) = f(t, x, x', x'')

Можна знизити порядок, ввівши нові змінні:
y₁ = x, y₂ = x', y₃ = x''

Тоді система стає:
y₁' = y₂
y₂' = y₃
y₃' = f(t, y₁, y₂, y₃)

Це дозволяє аналізувати рухи з урахуванням зміни прискорення (тобто ривку), що важливо для виявлення різких, неплавних рухів.

### Приклад 6: Використання системи ЛДР-1 для моделювання взаємодії між суглобами
Система лінійних диференціальних рівнянь першого порядку може моделювати взаємодію між кількома суглобами:
dx/dt = a₁₁x + a₁₂y + a₁₃z + f₁(t)
dy/dt = a₂₁x + a₂₂y + a₂₃z + f₂(t)
dz/dt = a₃₁x + a₃₂y + a₃₃z + f₃(t)

де x, y, z - відхилення від еталонної позиції для трьох суглобів, aᵢⱼ - коефіцієнти взаємодії, fᵢ(t) - зовнішні впливи.

Це дозволяє системі аналізувати, як зміна положення одного суглоба впливає на інші, що важливо для оцінки координації рухів під час виконання йога-вправ.

### Приклад 7: Застосування методу характеристик для аналізу динаміки
Для системи:
dX/dt = AX

де X = [x₁, x₂, ..., xₙ]ᵀ - вектор координат суглобів, A - матриця коефіцієнтів, розв'язок може бути знайдений за допомогою власних значень і власних векторів матриці A.

Якщо λᵢ - власні значення, vᵢ - власні вектори, то загальний розв'язок:
X(t) = Σ cᵢvᵢe^(λᵢt)

Це дозволяє аналізувати динаміку рухів і визначати, які рухові патерни є стійкими або нестійкими.

Ці математичні поняття дозволяють системі аналізувати складні рухові патерни, моделювати взаємодію між суглобами, передбачати динаміку рухів і забезпечувати ефективний аналіз якості виконання йога-вправ.
