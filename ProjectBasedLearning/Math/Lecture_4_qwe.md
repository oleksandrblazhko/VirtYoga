# Сценарії використання математичного опису для IT-проєкту "WebcamPhysExercise"

## Лекція 4. «Вектори. Означення вектору як напрямленого відрізку. Лінійні операції над векторами. Скалярний добуток векторів. Властивості скалярного добутку. Косинус кута між векторами. Направляючі косинуси»

### Обгрунтування користі:
Векторна алгебра є фундаментальним інструментом для аналізу просторових відносин між суглобами тіла людини. Використання векторів дозволяє точно визначати напрямки, відстані, кути між суглобами, що є критично важливим для аналізу йога-поз та надання рекомендацій щодо корекції рухів. Вектори є основою для аналітичної геометрії, що дозволяє описати будь-яку геометричну фігуру алгебраїчними засобами, що критично важливо для автоматичного аналізу пози тіла.

### Зв'язок математичної формалізації із описом IT-проєкту:
- Вектори використовуються для представлення напрямків між суглобами тіла
- Скалярний добуток векторів дозволяє обчислювати кути між суглобами
- Направляючі косинуси використовуються для аналізу орієнтації частин тіла
- Лінійні операції над векторами дозволяють моделювати рухи та трансформації пози
- Вектори дозволяють визначати геометричне місце точок (г. м. т.), що важливо для аналізу допустимих поз

### Приклади математичного опису IT-проєкту:

**1. Представлення напрямків між суглобами як векторів:**
```
Нехай A(x₁, y₁, z₁) і B(x₂, y₂, z₂) - координати двох суглобів.
Тоді вектор AB = (x₂-x₁, y₂-y₁, z₂-z₁) представляє напрямок від суглоба A до суглоба B.
Це дозволяє аналізувати положення кінцівок, тулуба та інших частин тіла.
```

**2. Обчислення кута між суглобами за допомогою скалярного добутку:**
```
Нехай AB = (x₁, y₁, z₁) і BC = (x₂, y₂, z₂) - вектори, що представляють напрямки
між трьома послідовними суглобами A, B, C.
Кут θ між цими векторами обчислюється за формулою:
cos(θ) = (AB · BC)/(|AB| × |BC|) = (x₁x₂ + y₁y₂ + z₁z₂)/(√(x₁²+y₁²+z₁²) × √(x₂²+y₂²+z₂²))

Це дозволяє визначати, наскільки точно користувач виконує певну йога-позу.
```

**3. Використання направляючих косинусів для аналізу орієнтації:**
```
Для вектора AB = (x, y, z) направляючі косинуси визначають кути, які вектор утворює з осями:
cos α = x/|AB|, cos β = y/|AB|, cos γ = z/|AB|

Ці значення дозволяють визначити, як саме орієнтована частина тіла (наприклад, рука або нога)
відносно координатних осей, що важливо для аналізу правильності пози.
```

**4. Лінійні операції над векторами для моделювання рухів:**
```
Сума векторів: якщо AB і BC - послідовні напрямки, то AC = AB + BC
Різниця векторів: вектор AD = AB - AC може представляти корекцію руху
Множення вектора на скаляр: k×AB змінює довжину вектора (наприклад, для масштабування руху)

Ці операції дозволяють моделювати можливі корекції пози для досягнення правильної йога-пози.
```

**5. Визначення відстані між суглобами:**
```
Відстань між двома суглобами A(x₁, y₁, z₁) і B(x₂, y₂, z₂) обчислюється як:
|AB| = √[(x₂-x₁)² + (y₂-y₁)² + (z₂-z₁)²]

Це дозволяє перевіряти, чи пропорції тіла в певній позі відповідають еталонним значенням.
```

**6. Визначення ортогональності (перпендикулярності) частин тіла:**
```
Два вектори AB і CD ортогональні, якщо їх скалярний добуток дорівнює нулю:
AB · CD = 0, тобто (x₁x₂ + y₁y₂ + z₁z₂) = 0

Це може бути корисним для аналізу, чи певні частини тіла (наприклад, тулуб і рука) 
розташовані перпендикулярно, як передбачено в певній йога-позі.
```

**7. Використання одиничних векторів (ортів) для нормалізації:**
```
Будь-який вектор AB можна нормалізувати, поділивши на його довжину:
e = AB/|AB| = (x/|AB|, y/|AB|, z/|AB|)

Це дає одиничний вектор у тому ж напрямку, що дозволяє порівнювати напрямки
без урахування довжини, що корисно для аналізу орієнтації частин тіла.
```

**8. Визначення проекції одного вектора на інший:**
```
Проекція вектора AB на вектор AC обчислюється як:
proj_AC(AB) = (AB · AC)/|AC|

Це дозволяє визначити, яка частина руху відбувається у напрямку еталонної пози.
```

**9. Визначення векторного рівняння прямої для моделювання ліній суглобів:**
```
Якщо A - початкова точка (суглоб), а s - напрямний вектор,
то векторне рівняння прямої: r = A + t×s, де t - параметр.

Це дозволяє моделювати лінію, утворену послідовністю суглобів.
```

**10. Використання векторів для визначення геометричного місця точок:**
```
Геометричне місце точок (г. м. т.), рівновіддалених від двох суглобів A і B,
може бути визначене як множина точок P, для яких |PA| = |PB|.
Це важливо для аналізу симетрії тіла.
```

Ці математичні моделі дозволяють системі аналізувати просторове розташування суглобів, обчислювати кути між ними, визначати орієнтацію частин тіла та надавати точні рекомендації щодо корекції пози для досягнення правильної йога-пози.
