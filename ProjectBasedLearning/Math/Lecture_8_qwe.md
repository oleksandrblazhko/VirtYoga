# Сценарії використання математичного опису для IT-проєкту "WebcamPhysExercise"

## Лекція 8. «Перетворення координат на площині. Канонічні рівняння кола, еліпса, гіперболи, параболи»

### Обгрунтування користі:
Перетворення координат дозволяє змінювати систему відліку для аналізу пози тіла, що особливо корисно при порівнянні з еталонними позами. Криві другого порядку можуть використовуватися для моделювання рухів тіла, траєкторій суглобів та аналізу амплітуди рухів, що важливо для оцінки правильності виконання йога-вправ. Аналітична геометрія, заснована на методі координат Рене Декарта, дозволяє описати будь-яку геометричну фігуру алгебраїчними засобами, що критично важливо для автоматичного аналізу пози тіла.

### Зв'язок математичної формалізації із описом IT-проєкту:
- Перетворення координат використовуються для зміни системи відліку відносно тіла користувача
- Канонічні рівняння кривих другого порядку застосовуються для моделювання траєкторій руху суглобів
- Кола та еліпси можуть моделювати обмеження на рухи певних частин тіла
- Перетворення координат дозволяють нормалізувати пози для порівняння з еталонними

### Приклади математичного опису IT-проєкту:

**1. Перетворення координат для нормалізації пози:**
```
Для порівняння пози користувача з еталонною, необхідно виконати перетворення координат:
- Перенесення: (x', y', z') = (x-x₀, y-y₀, z-z₀), де (x₀, y₀, z₀) - центр мас тіла
- Обертання: використання матриці обертання для вирівнювання тіла за еталонною орієнтацією
- Масштабування: (x'', y'', z'') = (kₓx', kᵧy', kᶻz'), для нормалізації розміру тіла

Це дозволяє порівнювати пози незалежно від положення, орієнтації та розміру користувача.
```

**2. Моделювання траєкторії руху суглоба за допомогою еліпса:**
```
Під час виконання певних йога-вправ суглоб може рухатися по еліптичній траєкторії:
(x-h)²/a² + (y-k)²/b² = 1

де (h, k) - центр еліпса, a і b - півосі, що визначають амплітуду руху.
Це може бути корисним для аналізу амплітуди руху руки або ноги під час виконання певної вправи.
```

**3. Визначення обмеженої області руху за допомогою кола:**
```
Обмеження на рух суглоба можна моделювати колом:
(x-h)² + (y-k)² ≤ r²

де (h, k) - центр області, r - радіус, що визначає максимально допустиму відстань від центральної точки.
Це може бути корисним для аналізу, чи суглоб залишається в межах безпечного діапазону руху.
```

**4. Використання параболи для моделювання траєкторії переходу між позами:**
```
Під час переходу між йога-позами суглоб може рухатися по параболічній траєкторії:
y = ax² + bx + c

або параметрично: x = t, y = at² + bt + c
Це дозволяє моделювати плавність переходу між позами та аналізувати ергономічність рухів.
```

**5. Перетворення системи координат для аналізу пози:**
```
Для аналізу пози тіла можна ввести нову систему координат, наприклад:
- Початок координат у центрі тазу для аналізу верхньої частини тіла
- Вісь X уздовж тулуба для аналізу симетрії
- Вісь Z уздовж вертикалі для аналізу балансу

Перетворення: [x']   [m₁₁ m₁₂ m₁₃] [x]   [tₓ]
              [y'] = [m₂₁ m₂₂ m₂₃] [y] + [tᵧ]
              [z']   [m₃₁ m₃₂ m₃₃] [z]   [t_z]

де M - матриця обертання, T - вектор перенесення.
```

**6. Використання гіперболи для аналізу екстремальних позицій:**
```
У деяких йога-вправах суглоби можуть досягати крайніх положень, які можна моделювати гіперболою:
x²/a² - y²/b² = 1

Це дозволяє аналізувати межі гнучкості та визначати, чи досягнуто максимально можливого розтягнення.
```

**7. Аналіз симетрії тіла за допомогою кривих другого порядку:**
```
Для аналізу симетрії тіла можна порівнювати траєкторії суглобів лівої та правої сторін тіла,
що можуть бути представлені як симетричні криві другого порядку.
Наприклад, якщо права рука рухається по еліпсу (x-h)²/a² + (y-k)²/b² = 1,
то ліва рука має рухатися по симетричному еліпсу відносно вертикалі.
```

**8. Визначення області допустимих поз тіла:**
```
Можна визначити область у просторі, де мають перебувати певні суглоби:
x²/a² + y²/b² + z²/c² ≤ 1 - еліпсоїд, що обмежує простір для певного суглоба.

Це дозволяє визначити, чи поза тіла відповідає анатомічно прийнятним обмеженням.
```

**9. Канонічне рівняння кола для аналізу циклічних рухів:**
```
Рівняння кола (x-x₀)² + (y-y₀)² = R² може моделювати циклічні рухи суглобів,
наприклад, обертання руки або ноги навколо суглоба під час виконання певної йога-вправи.
```

**10. Використання параметричного рівняння кола для аналізу руху:**
```
Коло можна задати параметрично:
x = x₀ + R·cos(t)
y = y₀ + R·sin(t)

де t - параметр часу, що дозволяє аналізувати рух суглоба по колу в залежності від часу.
Це корисно для аналізу плавності та ритму рухів.
```

Ці математичні моделі дозволяють системі аналізувати траєкторії руху, обмеження на рухи, симетрію тіла та нормалізувати пози для точного порівняння з еталонними йога-позами.
