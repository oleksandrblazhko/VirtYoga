# Сценарії використання математичного опису для IT-проєкту "WebcamPhysExercise"

## Лекція 9. «Числова послідовність. Границя числової послідовності. Теорема Вейєрштрасса»

### Обгрунтування користі:
Числові послідовності та їх границі є важливим інструментом для аналізу динаміки зміни пози тіла з часом. Це дозволяє відстежувати прогрес користувача, аналізувати стабільність пози, визначати тенденції покращення та оцінювати збіжність до еталонної пози. Особливо важливо для аналізу тривалості виконання пози та стійкості рівноваги. Теорема Вейєрштрасса гарантує існування границі для обмеженої монотонної послідовності, що критично важливо для аналізу прогресу користувача.

### Зв'язок математичної формалізації із описом IT-проєкту:
- Числові послідовності використовуються для представлення зміни параметрів пози з часом
- Границі послідовностей дозволяють аналізувати стабільність пози та збіжність до еталонної моделі
- Монотонність послідовностей використовується для аналізу прогресу користувача
- Теорема Вейєрштрасса застосовується для гарантування збіжності параметрів пози

### Приклади математичного опису IT-проєкту:

**1. Послідовність відхилень від еталонної пози:**
```
Нехай {dₙ} - послідовність відстаней від поточної пози до еталонної пози в моменти часу tₙ:
dₙ = ||P(tₙ) - P₀||, де P(tₙ) - поточна поза, P₀ - еталонна поза

Якщо lim(n→∞) dₙ = 0, то користувач збігається до еталонної пози.
Якщо lim(n→∞) dₙ = L > 0, то досягнуто стабільного відхилення.
```

**2. Аналіз стабільності пози за допомогою границі:**
```
Функція варіації пози: σ(t) = √[(1/n)∑(dᵢ(t) - d̄(t))²], де dᵢ(t) - відстань i-го суглоба від середнього.

Якщо lim(t→∞) σ(t) = 0, то поза стає все більш стабільною з часом.
Якщо lim(t→∞) σ(t) = const > 0, то досягнуто стабільного рівня варіації.
```

**3. Монотонна послідовність покращення:**
```
Послідовність {sₙ}, де sₙ - оцінка правильності пози в момент tₙ, може бути монотонною:
s₁ ≤ s₂ ≤ ... ≤ sₙ ≤ ...

Якщо послідовність обмежена зверху (sₙ ≤ S_max) і монотонна, то за теоремою Вейєрштрасса
вона має границю, що означає досягнення максимального рівня виконання пози.
```

**4. Визначення збіжності до еталонної пози:**
```
Послідовність {εₙ}, де εₙ - середнє відхилення суглобів від еталонної пози:
εₙ = (1/N)∑|Jᵢ(tₙ) - Jᵢ₀|, i = 1..N, де Jᵢ - координати i-го суглоба

Якщо lim(n→∞) εₙ = 0, то користувач навчився точно виконувати позу.
Швидкість збіжності може бути оцінена за формулою: εₙ ≈ C·αⁿ, 0 < α < 1
```

**5. Аналіз тривалості виконання пози:**
```
Послідовність {tₙ}, де tₙ - час утримання пози до n-го моменту:
t₁, t₂, ..., tₙ

Монотонне зростання цієї послідовності вказує на покращення витривалості.
Якщо послідовність обмежена, то досягнуто плато витривалості.
```

**6. Використання теореми Вейєрштрасса для аналізу прогресу:**
```
Якщо послідовність {pₙ}, що представляє рівень виконання певної йога-пози, є:
- обмеженою: a ≤ pₙ ≤ b для всіх n
- монотонною: pₙ ≤ pₙ₊₁ (наприклад, зростає з часом)

Тоді за теоремою Вейєрштрасса існує границя lim(n→∞) pₙ = p*, 
що означає досягнення стабільного рівня виконання пози.
```

**7. Аналіз коливань пози з часом:**
```
Послідовність {aₙ}, де aₙ - амплітуда коливань тіла навколо еталонної пози:
aₙ = max|P(t) - P₀| для t ∈ [tₙ, tₙ₊₁]

Якщо ця послідовність збігається до 0, то користувач навчився утримувати стабільну позу.
```

**8. Визначення швидкості навчання:**
```
Якщо sₙ - оцінка виконання пози в n-й сеанс, то швидкість навчання можна визначити як:
vₙ = sₙ₊₁ - sₙ

Якщо послідовність {vₙ} збігається до 0, то темп навчання зменшується, що свідчить про досягнення плато.
```

**9. Аналіз ефективності корекцій:**
```
Послідовність {cₙ}, де cₙ - покращення пози після корекції:
cₙ = ||P(tₙ₋₁) - P₀|| - ||P(tₙ) - P₀||

Якщо ця послідовність має границю, то можна оцінити ефективність корекцій.
```

**10. Визначення стабільного рівня виконання:**
```
Якщо послідовність {eₙ}, що представляє помилки виконання пози, є обмеженою та
відображає тенденцію до зменшення, то за теоремою Вейєрштрасса вона має границю,
що вказує на досягнення стабільного рівня виконання.
```

Ці математичні моделі дозволяють системі аналізувати динаміку виконання пози, відстежувати прогрес користувача, оцінювати стабільність та збіжність до еталонної моделі, а також прогнозувати майбутній прогрес.
