# Опис функціональних вимог до програми "Віртуальна Йога" (Virtual Yoga):
Опис використовує глосарій термінів із файлу Software\Docs\Glossary.md

## Прецедент 1 - Структурування опису йога-вправ
1) ФВ1.1 - користувач передає програмі неструктурований опис йога-вправ;
2) ФВ1.2 - програма структурує опис йога-вправ;
3) ФВ1.3 - програма передає структурованйи опис йога-вправи користувачу для узгодження;
4) ФВ1.4 - користувач за необхідністю виправляє структурований опис йога-вправи;
5) ФВ1.5 - користувач передає програмі виправлений структурований опис йога-вправи;
6) ФВ1.6 - програма зберігає структурований опис йога-вправи у базі даних

## Прецедент 2 - Підготовка до йога-вправи
1) ФВ2.1 - програма надає користувачу список йога-вправ;
2) ФВ2.2 - користувач обирає зі списку йога-вправу;
3) ФВ2.3 - програма вказує місце розташування вебкамери відносно користувача;
4) ФВ2.4 - програма передає користувачу опис обраної йога-вправи;

## Прецедент 3 - Виконанння йога-вправ
1) ФВ2.1 - програма передає користувачу опис йога-вправи;
2) ФВ2.2 - програма отримує положення тіла користувача;
3) ФВ2.3 - програма коментує правильне/неправильне положення тіла користувача із урахуванням опису йога-пози;
4) ФВ2.4 - програма надає рекомендації користувачу щодо напрямку зміни положення тіла користувача;
5) ФВ2.5 - програма повідомляє користувача про успішність виконання йога-пози;
