# Опис функціональних вимог до програми "Віртуальна Йога" (Virtual Yoga):
1) користувач стоїть перед вебкамерою, яка підключена до комп'ютера з програмою;
2) вебкамера розпізнає скелет користувача з використанням бібліотеки Mediapipe, яка містить 33 суглоби скелету користувача,
(https://ai.google.dev/edge/mediapipe/solutions/vision/pose_landmarker?hl=en):
  - перелік суглобів скелетону користувача (номер-назва):
  0-nose, 1-left eye, 2-left eye, 3-left eye, 4-right eye, 5-right eye, 6-right eye, 7-left ear, 8-right ear, 9-mouth (left), 10-mouth (right), 11-left shoulder, 
  12-right shoulder, 13-left elbow, 14-right elbow, 15-left wrist, 16-right wrist, 17-left pinky, 18-right pinky, 19-left index, 20-right index, 21-left thumb, 
  22-right thumb, 23-left hip, 24-right hip, 25-left knee, 26-right knee, 27-left ankle, 28-right ankle, 29-left heel, 30-right heel, 31-left foot index, 
  32-right foot index;
  - анатомічні зв'язки між суглобами скелетону (номери):
  0-4,4-5,5-6,6-8,0-1,1-2,2-3,3-7,10-9,18-20,18-16,20-16,22-16,16-14,14-12,19-17,15-19,15-17,21-15,13-21,
  11-13,12-11,24-23,12-24,24-26,26-28,28-32,28-30,32-30,11-23,23-25,25-27,27-31,27-29,29-31;
3) програма отримує дані координат (x,y,z) розташування суглобів;
4) користувач передає програмі неструктурований опис йога-вправ;
5) програма структурує опис йога-вправ;
6) програма передає структурованйи опис йога-вправи користувачу для узгодження;
7) користувач за необхідністю виправляє структурований опис йога-вправи;
8) користувач передає виправлений структурований опис йога-вправи до програми
9) програма зберігає структурований опис йога-вправи у базі даних:
- опис правильної пози;
- опис даних (x,y,z) та/або їх кутів про розташування суглобів правильної пози;
10) програма надає користувачу список йога-вправ;
11) користувач обирає зі списку йога-вправу;
12) програма передає користувачу опис обраної йога-вправи;
13) програма коментує правильне розташування суглобів із урахуванням опису йога-пози;
14) користувач стає у правильній йога-позі за рекомендаціями програми;
15) програма слідкує за зміною координат розташування суглобів користувача
та надає рекомендації щодо правильного напрямку зміни положення, щоб йога-поза користувача відповідала опису;
16) якщо користувач неправильно виконує рекомендації програми, наприклад, переміщує суглоби у неправильному напрямку,
програма повідомляє про це;
17) коли йога-поза користувача відповідає опису із урахуваннм припустимої похибки, 
програма повідомляє про успішність виконання йога-пози;
