# Опис функціональних вимог до програми "Віртуальна Йога" (Virtual Yoga):
Опис використовує глосарій термінів із файлу Software\Docs\Glossary.md

Прецедент 1 - Структурування опису йога-вправ
ФВ1.1) користувач передає програмі неструктурований опис йога-вправ;
ФВ1.2) програма структурує опис йога-вправ;
ФВ1.3) програма передає структурованйи опис йога-вправи користувачу для узгодження;
ФВ1.4) користувач за необхідністю виправляє структурований опис йога-вправи;
ФВ1.5) користувач передає виправлений структурований опис йога-вправи до програми
ФВ1.6) програма зберігає структурований опис йога-вправи у базі даних

Прецедент 2 - Підготовка до йога-вправи
ФВ2.1) програма надає користувачу список йога-вправ;
ФВ2.2) користувач обирає зі списку йога-вправу;
ФВ2.3) програма вказує місце розташування користувача відносно вебкамери;
ФВ2.4) програма передає користувачу опис обраної йога-вправи;

Прецедент 3 - Виконанння йога-вправ
ФВ2.1) програма передає користувачу опис йога-вправи;
ФВ2.2) програма отримує положення тіла користувача;
ФВ2.3) програма коментує правильне/неправильне положення тіла користувача із урахуванням опису йога-пози;
ФВ2.4) програма надає рекомендації користувачу щодо напрямку зміни положення тіла користувача;
ФВ2.5) програма повідомляє користувача про успішність виконання йога-пози;
